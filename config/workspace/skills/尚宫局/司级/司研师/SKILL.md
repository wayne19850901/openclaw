## 常见问题处理

### Q1: 如何判断是否触发子Agent？

**判断标准**：
- 简单研究 → 自身执行
- 复杂模型 → 触发外部工具
- 外部数据 → 触发neodata
- 跨局协调 → 触发上级

### Q2: 如何进行用户访谈？

**步骤**：
1. 确定访谈目标
2. 设计访谈提纲
3. 招募访谈对象
4. 执行深度访谈
5. 分析访谈数据
6. 提炼需求洞察
