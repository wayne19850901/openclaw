# 尚宫局工具依赖

## 核心工具

| 工具 | 用途 | 安装命令 |
|------|------|----------|
| Git | 版本控制 | `brew install git` |
| GitHub CLI | PR管理 | `brew install gh` |
| Bun | 脚本运行时 | `brew install oven-sh/bun/bun` |

## GitHub CLI配置

```bash
# 认证
gh auth login

# 验证
gh auth status
```

## 环境变量

```bash
# GitHub Token
GH_TOKEN=your_token
```
