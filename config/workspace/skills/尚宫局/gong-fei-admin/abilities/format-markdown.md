# 文档格式化

## 概述

格式化Markdown文档，统一文档风格。

## 使用方式

```bash
/baoyu-format-markdown path/to/document.md
/baoyu-format-markdown path/to/dir/ --recursive
```

## 格式化规则

| 规则 | 说明 |
|------|------|
| 标题层级 | 确保层级正确递进 |
| 代码块 | 添加语言标识 |
| 列表 | 统一列表格式 |
| 链接 | 检查链接有效性 |
| 图片 | 统一图片格式 |
| 表格 | 对齐表格列 |

## 输出

- 格式化后的文档覆盖原文件
- 同时输出格式化日志
