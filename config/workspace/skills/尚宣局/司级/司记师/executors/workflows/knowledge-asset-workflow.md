## 工作流概述

本工作流基于 Peter Lynch 的知识资产长期价值哲学，采用 **内容资产 + 数据资产 + 知识产权 + 品牌资产 + 用户资产** 五大核心方法论。

**执行原则**：
1. **强制读完机制**：AI必须执行完整流程，不得跳过任何步骤
2. **知识资产长期价值**：时间越长，知识资产价值越高
3. **知识资产复利效应**：知识积累产生复利
4. **知识资产护城河**：知识壁垒形成护城河

## Phase 1: 知识资产目标明确（强制执行）

### 1.1 确定知识资产管理目标

**任务**: 明确知识资产管理的目标。

**执行步骤**：
1. 明确知识资产管理目标：
   - 知识资产类型（内容/数据/专利/品牌/用户）
   - 知识资产管理目标
   - 知识资产时间期限

**输出格式**：
```
【知识资产管理目标】
- 知识资产类型：[类型]
- 管理目标：[目标描述]
- 时间期限：[期限]
```

**强制检查点**：必须明确知识资产管理目标，缺失不得进入下一阶段。

---

## Phase 2: 内容资产评估（强制执行）

### 2.1 内容资产数量评估

**任务**: 评估内容资产的数量。

**执行步骤**：
1. 统计内容资产数量：
   - 文章数量
   - 视频数量
   - 音频数量
   - 图片数量

**输出格式**：
```
【内容资产数量】
- 文章数量：[数量]篇
- 视频数量：[数量]个
- 音频数量：[数量]个
- 图片数量：[数量]张
- 内容资产总数：[总数]
```

**强制检查点**：
- 内容资产总数≥1000：进入下一阶段
- 内容资产总数<1000：**补充内容后继续**

---

### 2.2 内容资产质量评估

**任务**: 评估内容资产的质量。

**评分标准**（0-10分）：
- **9-10分**：内容数量≥10000篇，质量高，传播力强
- **7-8分**：内容数量≥5000篇，质量较高，传播力较强
- **5-6分**：内容数量≥1000篇，质量一般，传播力一般
- **3-4分**：内容数量<1000篇，质量较低，传播力较弱
- **0-2分**：内容资产失败，无内容资产

**输出格式**：
```
【内容资产质量】
- 内容质量评分：[评分]/10
- 内容传播力评分：[评分]/10
- 内容资产总分：[总分]/10
```

**强制检查点**：
- 内容资产总分≥6分：进入下一阶段
- 内容资产总分<6分：**立即终止，输出"不投资"决策**

## Phase 6: 最终决策（强制执行）

### 6.1 知识资产评分汇总

**任务**: 汇总所有评分，生成最终决策。

**评分权重**：
- 内容资产：25%
- 数据资产：25%
- 知识产权：20%
- 品牌资产：20%
- 用户资产：10%

**执行步骤**：
1. 计算综合评分（加权平均）
2. 给出最终决策（BUY/HOLD/SELL）
3. 给出信心水平（高/中/低）

**输出格式**：
```
【综合评分汇总】
评分项          得分    权重    加权得分
内容资产        [得分]  25%     [加权得分]
数据资产        [得分]  25%     [加权得分]
知识产权        [得分]  20%     [加权得分]
品牌资产        [得分]  20%     [加权得分]
用户资产        [得分]  10%     [加权得分]
------------------------------------------------
综合评分        [综合评分]（满分10分）

【最终决策】
- 决策：BUY / HOLD / SELL
- 信心水平：高 / 中 / 低
- 知识资产长期价值：[价值描述]
```

**强制检查点**：必须给出明确的 BUY/HOLD/SELL 决策。

---
