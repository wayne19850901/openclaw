> **约束**: 必须完整执行，不得跳过任何步骤

## 工作流概述

本工作流基于 Peter Lynch 的GARP投资哲学，采用 **PEG分析 + 十倍股筛选 + 行业熟悉度验证** 三大核心方法论。

**执行原则**：
1. **强制读完机制**：AI必须执行完整流程，不得跳过任何步骤
2. **数据源唯一性**：所有金融数据必须使用 `neodata-financial-search` skill
3. **量化指标验证**：每个决策必须有量化指标支撑（PEG≤1、行业熟悉度≥7分）
4. **十倍股验证**：必须验证十倍股潜力（5年增长≥10倍）

## Phase 1: 内容数据收集（强制执行）

### 1.1 获取目标公司内容相关数据

**任务**: 使用 `neodata-financial-search` skill 获取目标公司的内容相关数据。

**执行步骤**：
1. 调用 `neodata-financial-search` skill
2. 搜索目标公司的内容数据：
   - 内容生产量
   - 内容传播量
   - 用户互动率
   - 内容转化率
   - 内容ROI
   - 品牌曝光度

**输出格式**：
```
【内容数据】
- 内容生产量：[数量]篇/月
- 内容传播量：[数量]次/月
- 用户互动率：[互动率]%
- 内容转化率：[转化率]%

```

**强制检查点**：必须获取以上6项内容数据，缺失任何一项不得进入下一阶段。

## Phase 2: PEG分析（强制执行）

### 2.1 计算PEG比率

**任务**: 计算市盈率与增长率的比率，评估估值合理性。

**计算公式**：
```
PEG = 市盈率 / 净利润增长率
```

**评估标准**：
- **低估**：PEG≤0.5（极具投资价值）
- **合理**：PEG≤1.0（投资价值良好）
- **高估**：PEG>1.0（投资价值有限）

**输出格式**：
```
【PEG分析】
- 市盈率（PE）：[PE]倍
- 净利润增长率：[增长率]%
- PEG比率：[PEG]
- 评估结果：[低估/合理/高估]
- 投资价值：[价值描述]
```

**强制检查点**：必须计算PEG比率并给出评估结果。

## Phase 3: 十倍股潜力分析（强制执行）

### 3.1 评估十倍股潜力

**任务**: 评估目标公司是否具备十倍股潜力。

**评估维度**：
1. **市场规模**：TAM是否足够大（≥1000亿）
2. **竞争优势**：是否具备可持续竞争优势
3. **管理团队**：管理团队是否优秀且诚信
4. **财务健康**：财务状况是否健康
5. **增长动力**：是否具备持续增长动力

**评分标准**（0-10分）：
- **十倍股潜力**：总分≥8分
- **五倍股潜力**：总分≥7分
- **两倍股潜力**：总分≥6分
- **无潜力**：总分<6分

**输出格式**：
```
【十倍股潜力分析】
- 市场规模评分：[评分]/10
- 竞争优势评分：[评分]/10
- 管理团队评分：[评分]/10
- 财务健康评分：[评分]/10
- 增长动力评分：[评分]/10
- 十倍股潜力总分：[总分]/10
- 评估结果：[十倍股/五倍股/两倍股/无潜力]
```

**强制检查点**：必须评估十倍股潜力并给出评分。

---

## Phase 4: 行业熟悉度验证（强制执行）

### 4.1 评估行业熟悉度

**任务**: 评估对目标公司所在行业的熟悉程度。

**评估维度**：
1. **行业知识**：对行业基本知识的了解程度
2. **竞争格局**：对行业竞争格局的理解程度
3. **发展趋势**：对行业发展趋势的判断能力
4. **风险因素**：对行业风险因素的识别能力

**评分标准**（0-10分）：
- **高度熟悉**：熟悉度≥8分（可以重仓投资）
- **中度熟悉**：熟悉度≥7分（可以适度投资）
- **低度熟悉**：熟悉度≥6分（谨慎投资）
- **不熟悉**：熟悉度<6分（不建议投资）

**输出格式**：
```
【行业熟悉度验证】
- 行业知识评分：[评分]/10
- 竞争格局评分：[评分]/10
- 发展趋势评分：[评分]/10
- 风险因素评分：[评分]/10
- 行业熟悉度总分：[总分]/10
- 评估结果：[高度/中度/低度/不熟悉]
```

**强制检查点**：必须评估行业熟悉度并给出评分。

## Phase 5: 内容策略综合评分（强制执行）

### 5.1 计算内容策略总分

**任务**: 综合所有分析维度，计算内容策略总分。

**计算公式**：
```
内容策略总分 = (PEG评分 × 0.3) + (十倍股潜力评分 × 0.4) + (行业熟悉度 × 0.3)
```

**评分标准**（0-10分）：
- **优质策略**：内容策略总分≥8分
- **良好策略**：内容策略总分≥7分
- **一般策略**：内容策略总分≥6分
- **低质策略**：内容策略总分<6分

**输出格式**：
```
【内容策略综合评分】
- PEG评分：[评分]/10
- 十倍股潜力评分：[评分]/10
- 行业熟悉度评分：[评分]/10
- 内容策略总分：[总分]/10
- 评估结果：[优质/良好/一般/低质]
```

**强制检查点**：必须计算内容策略总分并给出评估结果。

## Phase 6: 最终决策与风险提示（强制执行）

### 6.1 综合评估与决策

**任务**: 综合所有分析结果，给出最终内容投资建议。

**决策标准**：
- **BUY（投资内容）**：内容策略总分≥8分、PEG≤1、十倍股潜力≥7分
- **HOLD（观察内容）**：内容策略总分≥7分、PEG≤1.5、十倍股潜力≥6分
- **SELL（放弃内容）**：内容策略总分<7分、PEG>1.5、十倍股潜力<6分

**输出格式**：
```
【最终决策】
- 决策：[BUY/HOLD/SELL]
- 内容策略总分：[总分]/10
- PEG比率：[PEG]
- 十倍股潜力评分：[评分]/10
- 核心理由：[理由描述]

【操作建议】
- 内容策略：[策略建议]
- 投资策略：[策略建议]
- 风险策略：[策略建议]
- 退出策略：[策略建议]
```

**强制检查点**：必须给出明确的BUY/HOLD/SELL决策和具体操作建议。

### 6.2 风险提示

**任务**: 提供内容投资的风险提示。

**输出格式**：
