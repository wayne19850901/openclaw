# PEG分析方法论

> **用途**: Peter Lynch GARP投资核心方法论
> **适用场景**: 估值合理性分析、增长型股票筛选
> **更新时间**: 2026-05-21

---

## 一、PEG定义

**PEG（Price/Earnings to Growth Ratio）**：市盈率与盈利增长率的比率，用于评估估值合理性。

**计算公式**：
```
PEG = 市盈率（PE） / 净利润增长率
```

**核心原则**：
- PEG≤1：估值合理或低估
- PEG>1：估值偏高
- 结合增长率判断估值

---

## 二、PEG计算方法

### 2.1 市盈率（PE）计算

**静态市盈率**：
```
PE（静态） = 当前股价 / 去年每股收益
```

**动态市盈率（TTM）**：
```
PE（TTM） = 当前股价 / 过去4个季度每股收益
```

**预测市盈率**：
```
PE（预测） = 当前股价 / 预测今年每股收益
```

---

### 2.2 净利润增长率计算

**历史增长率**：
```
增长率 = (今年净利润 - 去年净利润) / 去年净利润 × 100%
```

**复合增长率（CAGR）**：
```
CAGR = (期末净利润 / 期初净利润)^(1/n) - 1
```

**预测增长率**：
```
增长率 = (预测明年净利润 - 今年净利润) / 今年净利润 × 100%
```

---

## 三、PEG评估标准

### 3.1 基础评估标准

| PEG | 估值水平 | 投资建议 |
|-----|---------|----------|
| **≤0.5** | 显著低估 | 可以重仓投资 |
| **0.5-1.0** | 合理偏低 | 可以适度投资 |
| **1.0-1.5** | 合理偏高 | 谨慎投资 |
| **>1.5** | 显著高估 | 不建议投资 |

### 3.2 结合增长率评估

**高增长率（>30%）**：
- PEG≤1.5：可接受
- PEG>1.5：估值偏高

**中增长率（15-30%）**：
- PEG≤1.0：可接受
- PEG>1.0：估值偏高

**低增长率（<15%）**：
- PEG≤0.8：可接受
- PEG>0.8：估值偏高

## 四、PEG分析案例

### 4.1 案例：贵州茅台

**数据**：
- 当前股价：1,311元
- 每股收益（TTM）：66.0元
- 净利润增长率：1.47%（2026一季报）

**计算**：
```
PE = 1,311 / 66.0 = 19.85倍
增长率 = 1.47%
PEG = 19.85 / 1.47 = 13.5
```

**评估结果**：
- PEG=13.5，显著高估（>1.5）
- 原因：净利润增长率过低（1.47%）
- 建议：HOLD（等待增长恢复）

---

### 4.2 案例：宁德时代

**数据**：
- 当前股价：411.63元
- 每股收益（TTM）：17.1元
- 净利润增长率：约20%（预估）

**计算**：
```
PE = 411.63 / 17.1 = 24.07倍
增长率 = 20%
PEG = 24.07 / 20 = 1.20
```

**评估结果**：
- PEG=1.20，合理偏高（1.0-1.5）
- 原因：增长率较高（20%）
- 建议：HOLD（观察增长持续性）

---

### 4.3 案例：某高增长公司

**数据**：
- 当前股价：100元
- 每股收益（TTM）：2元
- 净利润增长率：50%

**计算**：
```
PE = 100 / 2 = 50倍
增长率 = 50%
PEG = 50 / 50 = 1.0
```

**评估结果**：
- PEG=1.0，合理偏低（≤1.0）
- 原因：高增长率（50%）支撑估值
- 建议：BUY（可以适度投资）

## 五、十倍股筛选标准

### 5.1 十倍股特征

**Peter Lynch十倍股筛选标准**：

1. **PEG≤1.0**：估值合理
2. **增长率≥20%**：高增长
3. **行业熟悉度≥7分**：熟悉的行业
4. **护城河≥6分**：有竞争优势
5. **财务健康≥7分**：财务状况良好
6. **管理团队≥7分**：管理团队优秀
7. **市场规模≥100亿**：市场空间大
8. **竞争格局良好**：行业前三
9. **政策支持**：政策支持力度大
10. **技术优势**：技术领先

### 5.2 十倍股评分

**评分公式**：
```
十倍股潜力总分 = (PEG评分 × 0.2) + (增长率评分 × 0.3) + (行业熟悉度 × 0.2) + (护城河 × 0.2) + (财务健康 × 0.1)
```

**评分标准**（0-10分）：
- **≥8分**：十倍股潜力
- **7-8分**：五倍股潜力
- **6-7分**：两倍股潜力
- **<6分**：无潜力

## 六、最佳实践

### 6.1 PEG分析流程

1. **计算PE**：使用TTM市盈率
2. **计算增长率**：使用历史或预测增长率
3. **计算PEG**：PE / 增长率
4. **评估合理性**：结合增长率判断
5. **给出建议**：BUY/HOLD/SELL

### 6.2 常见误区

1. **过度依赖PEG**：PEG是参考指标，需结合其他因素
2. **忽视增长率可持续性**：高增长可能无法持续
3. **忽视行业差异**：不同行业PEG标准不同
4. **忽视周期性**：周期性行业PEG波动大
