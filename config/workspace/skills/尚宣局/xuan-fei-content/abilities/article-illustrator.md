# 文章配图

## 概述

为文章自动生成配图，增强内容的视觉吸引力。

## 使用方式

```bash
/baoyu-article-illustrator path/to/article.md
```

## 工作流程

1. **分析文章**：识别关键段落和配图机会
2. **生成提示词**：为每个配图位置创建描述
3. **生成图片**：调用图片生成API
4. **插入文章**：将图片插入到合适位置

## 配图类型

| 类型 | 适用场景 |
|------|----------|
| 封面图 | 文章开头 |
| 概念图 | 抽象概念解释 |
| 流程图 | 步骤说明 |
| 数据图 | 数据展示 |
| 插图 | 章节分隔 |

## 输出

```
{article-dir}/
├── article.md           # 原文章
├── article-illustrated.md  # 配图后文章
└── images/
    ├── cover.png
    ├── concept-1.png
    └── ...
```
