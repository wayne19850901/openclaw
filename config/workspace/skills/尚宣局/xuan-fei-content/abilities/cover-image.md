# 封面图生成

## 概述

为文章、视频、社交媒体内容生成专业封面图。

## 使用方式

```bash
/baoyu-cover-image --title "文章标题" --style minimal
/baoyu-cover-image path/to/article.md
```

## 封面风格

| 风格 | 适用场景 |
|------|----------|
| `minimal` | 极简风格 |
| `gradient` | 渐变背景 |
| `photo` | 照片叠加 |
| `illustration` | 插画风格 |
| `typography` | 字体设计 |

## 尺寸预设

| 平台 | 尺寸 |
|------|------|
| 微信公众号 | 900×500 |
| 小红书 | 1242×1660 |
| 知乎 | 1200×675 |
| 视频封面 | 1920×1080 |
