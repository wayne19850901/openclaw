# AI绘图

## 概述

通用AI绘图能力，根据文字描述生成图片。

## 使用方式

```bash
/baoyu-image-gen "一个未来城市的概念图"
/baoyu-image-gen path/to/prompt.md --style cyberpunk
```

## 风格预设

| 风格 | 描述 |
|------|------|
| `realistic` | 写实 |
| `artistic` | 艺术风格 |
| `anime` | 动漫 |
| `cyberpunk` | 赛博朋克 |
| `watercolor` | 水彩 |
| `oil-painting` | 油画 |

## 参数选项

| 参数 | 说明 |
|------|------|
| `--aspect` | 宽高比（16:9, 1:1, 9:16） |
| `--style` | 风格预设 |
| `--quality` | 质量等级 |
| `--count` | 生成数量 |
