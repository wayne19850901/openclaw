# 20 种视觉风格

## 简约风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **极简** | 大留白、少元素、黑白灰 | 高端品牌、科技产品 |
| **北欧** | 自然色、柔和、温馨 | 家居、生活方式 |

| 风格 | 特点 | 适用场景 |

| **现代商务** | 蓝色系、几何、简洁 | 企业介绍、产品发布 |
