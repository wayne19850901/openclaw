# 图片生成

## 概述

生成图片和插图。支持多种风格和尺寸。

**来源**: 尚宣局/xuan-fei-content

## 使用方式

```bash
/image-gen [描述] [--style <风格>] [--size <尺寸>]
```

## 工作流程

```
1. 解析描述和需求
2. 选择风格
3. 选择尺寸
4. 生成图片
5. 输出文件
```

## 尺寸选项

| 尺寸 | 用途 |
|------|------|
| **1024x1024** | 社交媒体 |
| **1920x1080** | 横幅、封面 |
| **800x600** | 文章配图 |
| **400x400** | 头像、图标 |
