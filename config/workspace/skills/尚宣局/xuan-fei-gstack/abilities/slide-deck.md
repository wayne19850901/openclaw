# PPT 生成

## 概述

生成演示文稿和幻灯片。支持多种模板和风格。

**来源**: 尚宣局/xuan-fei-content

## 使用方式

```bash
/ppt [主题] [--template <模板>] [--slides <数量>]
```

## 工作流程

```
1. 解析主题和内容
2. 规划幻灯片结构
3. 选择模板
4. 生成 HTML/PPTX
5. 输出文件
```

## 模板类型

| 模板 | 适用场景 |
|------|----------|
| **商务汇报** | 数据展示、业绩报告 |
| **产品介绍** | 功能演示、价值主张 |
| **培训课件** | 教学内容、知识点 |
| **项目提案** | 方案展示、可行性分析 |
| **故事叙述** | 情感共鸣、品牌故事 |
