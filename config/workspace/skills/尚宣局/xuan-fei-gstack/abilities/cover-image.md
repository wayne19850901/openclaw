# 封面设计

## 概述

设计封面图和封面页。适用于文章、报告、书籍。

**来源**: 尚宣局/xuan-fei-content

## 使用方式

```bash
/cover [主题] [--type <类型>] [--style <风格>]
```

## 工作流程

```
1. 解析主题和类型
2. 选择风格
3. 设计布局
4. 生成封面
5. 输出文件
```

## 类型选项

| 类型 | 特点 |
|------|------|
| **文章封面** | 标题 + 配图 |
| **报告封面** | 标题 + 数据可视化 |
| **书籍封面** | 标题 + 作者 + 插图 |
