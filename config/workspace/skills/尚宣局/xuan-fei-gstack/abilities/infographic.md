# 信息图生成

## 概述

将数据和信息转化为可视化信息图。21 种布局模板，20 种视觉风格。

**来源**: 尚宣局/xuan-fei-content

## 使用方式

```bash
/infographic [主题] [--layout <布局>] [--style <风格>]
```

## 工作流程

```
1. 解析主题和数据
2. 选择布局模板
3. 选择视觉风格
4. 生成 SVG/HTML
5. 输出文件
```

## 布局模板（21 种）

| 类别 | 布局 |
|------|------|
| **对比型** | 左右对比、上下对比、前后对比 |
| **流程型** | 线性流程、循环流程、分支流程 |
| **层级型** | 树状图、金字塔、同心圆 |
| **数据型** | 柱状图、饼图、折线图、雷达图 |
| **地图型** | 地理分布、热力图、路线图 |
| **时间型** | 时间轴、里程碑、甘特图 |

## 视觉风格（20 种）

| 风格 | 特点 |
|------|------|
| **简约** | 极简、留白多 |
| **商务** | 专业、数据驱动 |
| **科技** | 蓝色系、未来感 |
| **自然** | 绿色系、有机形态 |
| **复古** | 经典、怀旧 |
| **现代** | 大胆配色、几何 |
| **插画** | 手绘风格 |
| **3D** | 立体效果 |
