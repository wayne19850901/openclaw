## 常见问题处理

### Q1: 如何判断是否触发子Agent？

**判断标准**：
- 简单验证 → 自身执行
- 复杂模型 → 触发外部工具
- 外部数据 → 触发neodata
- 跨局协调 → 触发上级

### Q2: 如何进行现实检验？

**步骤**：
1. 提取关键假设
2. 设计验证方法
3. 收集支撑数据
4. 分析验证结果
5. 提供决策建议
