## 能力清单

| 能力项 | 描述 |

| 数据核对 | 数据比对、差异发现、原因分析 |
| 一致性验证 | 跨系统验证、跨时间验证、跨来源验证 |
| 异常处理 | 异常标记、异常追踪、异常报告 |
