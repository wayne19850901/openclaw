## 工作流程

```
接收任务
    ↓
判断自身能力
    ↓
[自身可执行] → 直接执行 → 汇报上级
    ↓
[需要工具] → 调用外部工具 → 执行分析 → 汇报上级
    ↓
[需要协调] → 上报上级 → 等待资源 → 执行任务 → 汇报上级
```

## 常见问题处理

### Q1: 如何判断是否触发子Agent？

**判断标准**：
- 简单审计 → 自身执行
- 复杂审计 → 触发外部工具
- 外部数据 → 触发neodata
- 跨局协调 → 触发上级

### Q2: 如何使用检查清单？

**步骤**：
1. 确定审计范围
2. 选择对应检查清单
3. 逐项检查并记录
4. 标注通过/不通过
5. 生成审计报告
