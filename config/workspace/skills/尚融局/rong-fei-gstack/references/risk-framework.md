# 风险评估框架

## 风险分类

### 信用风险

| 类型 | 指标 | 预警阈值 |
|------|------|----------|
| 违约风险 | 违约概率(PD) | >5% |
| 损失风险 | 违约损失率(LGD) | >40% |
| 集中风险 | 单一客户占比 | >10% |

### 市场风险

| 类型 | 指标 | 说明 |
|------|------|------|
| 利率风险 | 久期、凸性 | 利率敏感性 |
| 汇率风险 | 外汇敞口 | 汇率波动影响 |
| 价格风险 | VaR | 最大可能损失 |

### 操作风险

| 类型 | 示例 | 防范措施 |
|------|------|----------|
| 流程风险 | 操作失误 | 流程标准化 |
| 系统风险 | IT故障 | 备份与恢复 |
| 人员风险 | 欺诈、离职 | 内控制度 |
| 外部风险 | 自然灾害 | 保险、应急预案 |

## 风险评估矩阵

```
         │ 低影响 │ 中影响 │ 高影响
─────────┼────────┼────────┼────────
高概率   │  监控  │  缓解  │  优先处理
中概率   │  接受  │  监控  │  缓解
低概率   │  接受  │  接受  │  监控
```

## 风险指标

| 风险等级 | 颜色 | 行动 |
|----------|------|------|
| 极高 | 🔴 | 立即处理 |
| 高 | 🟠 | 优先处理 |
| 中 | 🟡 | 制定计划 |
| 低 | 🟢 | 持续监控 |
