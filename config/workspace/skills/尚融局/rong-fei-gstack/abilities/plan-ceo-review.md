# CEO 计划评审

## 概述

CEO/创始人级别的计划评审技能。四种模式应对不同战略情境，11 个评审章节覆盖架构、安全、测试、性能、部署、长期轨迹等维度。

**来源**: gstack/plan-ceo-review

## 使用方式

```bash
/ceo-review <plan-file> [--mode expansion|selective|hold|reduction]
```

## 四种模式

| 模式 | 触发条件 | 核心问题 |
|------|----------|----------|
| **EXPANSION** | 野心功能、新方向 | "是否足够大？" |
| **SELECTIVE** | 有诱惑力的扩展 | "是否正确 + 还有什么值得？" |
| **HOLD** | 范围合理 | "是否过于复杂？" |
| **REDUCTION** | 范围过大、时间紧 | "是否是最低限度？" |

## 工作流程

```
Step 0: 系统审计
    ├── 读取 CLAUDE.md（工程偏好）
    ├── 读取 DESIGN.md（设计系统）
    └── 读取 TODOS.md（待办事项）
    ↓
Step 0D: 模式选择
    ├── 用户选择
    └── 或 AI 推荐
    ↓
Section 1-11: 评审章节
    ├── 1. 架构评审
    ├── 2. 错误处理评审
    ├── 3. 安全评审
    ├── 4. 数据/UX 边界评审
    ├── 5. 代码质量评审
    ├── 6. 测试评审
    ├── 7. 性能评审
    ├── 8. 可观测性评审
    ├── 9. 部署评审
    ├── 10. 长期轨迹评审
    └── 11. 设计/UX 评审
    ↓
外部意见（可选）
    ├── Codex 执行
    └── 或 Claude 子代理
    ↓
输出
    ├── CEO 计划文档
    ├── 实现任务列表
    └── 评审报告
```

## 执行指令

**加载原始 gstack SKILL.md**:

```bash
# 读取完整技能定义
source_content=$(cat ~/.claude/skills/gstack/plan-ceo-review/SKILL.md)
```

将 `source_content` 作为系统提示词的一部分，执行完整的 CEO 评审流程。

## 评审章节速查

| 章节 | 关注点 | EXPANSION 特有 |
|------|--------|----------------|
| 1. 架构 | 模块边界、依赖方向 | 平台潜力评估 |
| 2. 错误处理 | 异常映射、救援路径 | 混沌场景 |
| 3. 安全 | 认证、授权、注入 | — |
| 4. 数据/UX | 边界值、并发 | — |
| 5. 代码质量 | 命名、重复 | — |
| 6. 测试 | 金字塔、覆盖率 | 混沌测试 |
| 7. 性能 | N+1、索引、缓存 | — |
| 8. 可观测性 | 日志、指标、追踪 | 运维愉悦度 |
| 9. 部署 | 迁移、回滚 | 部署基础设施 |
| 10. 长期轨迹 | 技术债、可逆性 | Phase 2/3 规划 |
| 11. 设计/UX | 信息架构、交互状态 | UI 不可避免性 |

```markdown
{expanded vision if EXPANSION}

## Scope Decisions

- {item 1}
- {item 2}

### Deferred to TODOS.md
- {item 1}

### Skipped
- {item 1}

## Architecture Decisions
{key decisions from review}

## Risk Profile
{changes to deployment risk}

## Next Steps
{recommended next reviews}
```

### 实现任务列表

```markdown
## Implementation Tasks

- [ ] **T1 (P1)** — {component} — {title}
  - Surfaced by: Section {N}
  - Files: {paths}
  - Verify: {test command}
```

## 禁令

- **禁止修改代码** — 评审技能，不写实现
- **一次一问** — 每个发现单独确认
- **必须裁决** — 研究主管必须给出明确 Buy/Sell/Hold
