## 工作流概述

本工作流基于 Warren Buffett 的价值投资哲学，采用 **护城河分析 + 安全边际评估 + 长期持有策略** 三大核心方法论。

**执行原则**：
1. **强制读完机制**：AI必须执行完整流程，不得跳过任何步骤
2. **数据源唯一性**：所有金融数据必须使用 `neodata-financial-search` skill
3. **量化指标验证**：每个决策必须有量化指标支撑（ROE≥15%、买入价格≤内在价值×70%）
4. **护城河评分**：必须完成护城河评分（0-10分），低于6分不得买入

## Phase 2: 护城河分析（强制执行）

### 2.1 护城河五大要素评分

**任务**: 基于 Warren Buffett 的护城河理论，对目标公司进行五大要素评分。

**护城河五大要素**：
1. **无形资产**（品牌、专利、牌照）
2. **转换成本**（客户更换供应商的成本）
3. **网络效应**（用户越多价值越大）
4. **成本优势**（规模效应、独特资源）
5. **有效规模**（市场容量限制）

**评分标准**（每个要素0-10分）：
- 0-3分：无护城河
- 4-6分：弱护城河
- 7-9分：强护城河
- 10分：极强护城河

**执行步骤**：
1. 分析每个护城河要素的证据
2. 给出量化评分（0-10分）
3. 计算综合护城河评分（加权平均）

**输出格式**：
```
【护城河评分】
要素              评分(0-10)    证据分析
无形资产          [评分]        [分析]
转换成本          [评分]        [分析]
网络效应          [评分]        [分析]
成本优势          [评分]        [分析]
有效规模          [评分]        [分析]
------------------------------------------------
综合护城河评分    [综合评分]    （加权平均）
```

**强制检查点**：
- 综合护城河评分 ≥ 6分：进入下一阶段
- 综合护城河评分 < 6分：**立即终止分析，输出"不买入"决策**

## Phase 3: 安全边际评估（强制执行）

### 3.1 内在价值估算（DCF法）

**任务**: 使用 DCF（现金流折现）法估算内在价值。

**执行步骤**：
1. 基于过去5年的 FCF 数据
2. 预测未来10年的 FCF 增长率
3. 计算内在价值（折现率10%）

**计算公式**：
```
内在价值 = Σ(未来FCF / (1+折现率)^年份) + 残值
```

**输出格式**：
```
【内在价值估算】
- 过去5年平均FCF：[平均FCF]亿元
- 预测未来10年FCF增长率：[增长率]%
- 折现率：10%
- 计算内在价值：[内在价值]亿元
- 每股内在价值：[每股内在价值]元
```

**强制检查点**：必须完成内在价值计算，缺失不得进入下一阶段。

---

### 3.2 安全边际计算

**任务**: 计算安全边际（买入价格上限）。

**安全边际标准**：
- **买入价格 ≤ 内在价值 × 70%**（Warren Buffett标准）
- 安全边际 ≥ 30%

**执行步骤**：
1. 计算买入价格上限（内在价值 × 70%）
2. 对比当前股价
3. 计算安全边际百分比

**输出格式**：
```
【安全边际评估】
- 内在价值：[内在价值]元/股
- 买入价格上限（内在价值×70%）：[上限]元/股
- 当前股价：[股价]元/股
- 安全边际：[安全边际]%
```

**强制检查点**：
- 安全边际 ≥ 30%：进入下一阶段
- 安全边际 < 30%：**立即终止分析，输出"不买入"决策**

## Phase 4: ROE验证（强制执行）

### 4.1 ROE历史分析

**任务**: 验证ROE是否满足 Warren Buffett 标准。

**ROE标准**：
- **过去5年平均ROE ≥ 15%**
- ROE稳定性（波动不超过5%）

**执行步骤**：
1. 计算过去5年平均ROE
2. 计算ROE波动率
3. 对比标准

**输出格式**：
```
【ROE验证】
- 过去5年平均ROE：[平均ROE]%
- ROE波动率：[波动率]%
- 是否满足标准（≥15%）：[是/否]
```

**强制检查点**：
- 平均ROE ≥ 15%：进入下一阶段
- 平均ROE < 15%：**立即终止分析，输出"不买入"决策**

**输出格式**：
