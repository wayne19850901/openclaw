> **更新时间**: 2026-05-21

---

## 一、LTV/CAC定义

**LTV（Lifetime Value，客户生命周期价值）**：单个客户在整个生命周期内为企业创造的总价值。

**CAC（Customer Acquisition Cost，获客成本）**：获取单个客户所需的平均成本。

**LTV/CAC比率**：衡量获客效率的核心指标，反映获客投资回报率。

---

## 二、LTV计算方法

### 2.1 基础公式

```
LTV = ARPU × 客户生命周期长度
```

其中：
- ARPU（Average Revenue Per User）：平均每用户收入
- 客户生命周期长度：客户平均留存时间（月或年）

### 2.2 详细公式

```
LTV = (ARPU × 毛利率) / 客户流失率
```

**案例计算**：
- ARPU：1,000元/月
- 毛利率：80%
- 客户流失率：5%/月

```
LTV = (1,000 × 0.8) / 0.05 = 800 / 0.05 = 16,000元
```

---

## 三、CAC计算方法

### 3.1 基础公式

```
CAC = 总获客成本 / 新增客户数量
```

### 3.2 总获客成本包含

1. **营销成本**：广告投放、内容营销、SEO等
2. **销售成本**：销售人员薪资、提成、差旅等
3. **技术成本**：营销工具、CRM系统等
4. **管理成本**：营销团队管理费用

**案例计算**：
- 月营销成本：100万元
- 月销售成本：50万元
- 月新增客户：500人

```
CAC = (100万 + 50万) / 500 = 150万 / 500 = 3,000元/人
```

---

## 四、LTV/CAC比率评估

### 4.1 评估标准

| LTV/CAC | 获客效率 | 投资建议 |
|---------|---------|----------|
| **≥3:1** | 优秀 | 可以加大获客投入 |
| **2:1-3:1** | 良好 | 维持当前获客策略 |
| **1:1-2:1** | 一般 | 需优化获客策略 |
| **<1:1** | 不足 | 需重新评估获客模式 |

### 4.2 案例分析

**案例1：SaaS公司**
- LTV：16,000元
- CAC：3,000元
- LTV/CAC：16,000 / 3,000 = 5.33:1

**评估结果**：优秀（≥3:1），可以加大获客投入

**案例2：电商公司**
- LTV：800元
- CAC：500元
- LTV/CAC：800 / 500 = 1.6:1

**评估结果**：一般（1:1-2:1），需优化获客策略

## 五、逆向思维获客验证

### 5.1 失败场景分析

**场景1：获客渠道失效**
- 问题：主要获客渠道（如广告）效果下降
- 应对：多元化获客渠道，降低单一渠道依赖

**场景2：客户流失率上升**
- 问题：客户留存率下降，LTV降低
- 应对：提升产品质量和客户服务，降低流失率

**场景3：获客成本上升**
- 问题：竞争加剧，CAC上升
- 应对：优化获客效率，提升转化率

**场景4：竞争对手获客更优**
- 问题：竞争对手LTV/CAC更高
- 应对：分析竞争对手策略，学习最佳实践

## 六、最佳实践

### 6.1 提升LTV的方法

1. **提升ARPU**：
   - 增加产品功能和服务
   - 实施差异化定价策略
   - 推出高价值增值服务

2. **延长客户生命周期**：
   - 提升产品质量和用户体验
   - 加强客户服务和客户成功
   - 建立客户忠诚度计划

3. **降低客户流失率**：
   - 识别流失风险客户
   - 提前干预和挽留
   - 优化产品和服务

### 6.2 降低CAC的方法

1. **优化获客渠道**：
   - 分析各渠道ROI
   - 关停低效渠道
   - 加大高效渠道投入

2. **提升转化率**：
   - 优化落地页和注册流程
   - A/B测试优化
   - 提升销售团队效率

3. **降低获客成本**：
   - 利用自然流量（SEO、口碑）
   - 实施推荐奖励计划
   - 优化广告投放效率

## 七、案例库

### 7.1 优秀案例

**Netflix**
- LTV/CAC：约4:1
- 策略：高质量内容+个性化推荐+低流失率

**Amazon Prime**
- LTV/CAC：约5:1
- 策略：会员权益+快速配送+高复购率

### 7.2 失败案例

**某O2O公司**
- LTV/CAC：<1:1
- 问题：获客成本过高，客户流失率过高
- 结果：资金链断裂，公司倒闭
