# intelligence — 情报搜集核心能力

## 触发词
- "情报奏折"、"搜集情报"、"监控"
- "daily scan"、"今日情报"

## 核心功能
四大领域情报奏折：
- 💰 金融（贷款/投资）
- 🤖 科技（AI应用）
- 📱 媒体（自媒体运营）
- 📚 教育（应试指南）

## 禁令体系

```markdown
【禁令】未核实信源前禁止发布情报
【禁令】禁止引用单一来源做出结论
【禁令】禁止编造或推测无数据支撑结论
【禁令】热点描述禁止超过一句话
```

## 热点表达规范

### ✅ 正确示例
```
1、央行宣布2026年降准降息，1年期LPR维持3.0%；
2、六大行金融科技投入超1300亿元，AI风控进入规模化落地；
```

### ❌ 错误示例
```
1、央行在今天的发布会上宣布了一个非常重要的消息...（太长）
```
