## 工具优先级

```
news-aggregator-skill（首选）→ web-search-tavily（补充）→ DuckDuckGo（兜底）
```

### news-aggregator-skill

**覆盖范围**：28个情报源（Hacker News、GitHub、Hugging Face、WallStreetCN、微博、Podcasts等）

**触发命令**：
```
如意如意        # 打开交互式菜单
每日扫描        # 执行全源扫描
技术新闻        # Tech 域专项
财经更新        # Finance 域专项
AI 简报         # AI 论文专项
```

### web-search-tavily

**命令模式**：
```bash
# 基础搜索
tavily search "关键词"

# 新闻搜索（Finance/Tech）
tavily search "关键词" --topic news --time day --answer

# 金融搜索
tavily search "股票代码" --topic finance
```

## 禁止系统

1. ❌ 禁止采集政治敏感内容（中国政治人物、政府机构、政策批判）
2. ❌ 禁止采集个人隐私信息
3. ❌ 禁止生成虚假新闻或误导性内容
4. ❌ 禁止对政治事件发表评论或分析

## 热点表达规范

```
✅ 正确：据[来源]报道，[事件]引发关注
❌ 错误：[事件]说明[政治观点]
```

**中性表述**：客观陈述事实，引用来源，不做主观评价。

## 情报分级

| 级别 | 定义 | 处理方式 |
|------|------|----------|
| P1 | 紧急情报（重大事件） | 立即上报 |
| P2 | 重要情报（趋势变化） | 定时汇总 |
| P3 | 一般情报（日常动态） | 批量归档 |

## 注意事项

- **来源标注**：所有情报必须标注来源
- **时效性**：实时情报需标注采集时间
- **交叉验证**：重要情报需多源验证
- **保密性**：敏感情报需加密存储
