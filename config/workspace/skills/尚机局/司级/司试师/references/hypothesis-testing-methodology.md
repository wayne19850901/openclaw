# 假设测试方法论

> **用途**: 科学方法论核心方法论
> **适用场景**: 假设驱动、数据验证、实验设计
> **更新时间**: 2026-05-21

---

## 一、假设测试定义

**假设测试（Hypothesis Testing）**：科学方法论的核心方法，通过提出假设、设计实验、收集数据、验证假设，得出科学结论。

**核心原则**：
- 假设驱动（先提出假设，再验证假设）
- 数据验证（用数据验证假设）
- 可重复性（实验结果可重复）
- 可证伪性（假设可被证伪）

---

## 二、假设测试要素

### 2.1 假设提出（Hypothesis Formulation）

**定义**：提出可验证的假设，作为实验的起点。

**评估指标**：
- 假设清晰度（假设表述清晰）
- 假设可验证性（假设可被验证）
- 假设可证伪性（假设可被证伪）

**评分标准**（0-10分）：
- **9-10分**：假设清晰、可验证、可证伪
- **7-8分**：假设较清晰、可验证、可证伪
- **5-6分**：假设一般、可验证、可证伪
- **3-4分**：假设不清晰、难以验证、难以证伪
- **0-2分**：假设失败，无法验证

**案例**：
- **投资假设**：假设提出9/10，假设清晰、可验证、可证伪
- **产品假设**：假设提出8/10，假设较清晰、可验证、可证伪

---

### 2.2 实验设计（Experimental Design）

**定义**：设计科学实验，验证假设。

**评估指标**：
- 实验变量控制（控制变量、实验变量）
- 实验样本量（样本量≥30）
- 实验随机性（随机分组）

**评分标准**（0-10分）：
- **9-10分**：实验设计科学，变量控制严格，样本量充足
- **7-8分**：实验设计良好，变量控制较好，样本量充足
- **5-6分**：实验设计一般，变量控制一般，样本量充足
- **3-4分**：实验设计较差，变量控制不足，样本量不足
- **0-2分**：实验设计失败，无法验证假设

**案例**：
- **A/B测试**：实验设计9/10，实验设计科学
- **产品测试**：实验设计8/10，实验设计良好

---

### 2.3 数据收集（Data Collection）

**定义**：收集实验数据，用于验证假设。

**评估指标**：
- 数据质量（数据准确性、数据完整性）
- 数据量（数据量≥30个样本）
- 数据多样性（数据来源多样性）

**评分标准**（0-10分）：
- **9-10分**：数据质量高，数据量充足，数据来源多样
- **7-8分**：数据质量较高，数据量充足，数据来源较多样
- **5-6分**：数据质量一般，数据量充足，数据来源一般
- **3-4分**：数据质量较低，数据量不足，数据来源单一
- **0-2分**：数据收集失败，无法验证假设

**案例**：
- **投资数据**：数据收集9/10，数据质量高
- **产品数据**：数据收集8/10，数据质量较高

---

### 2.4 统计分析（Statistical Analysis）

**定义**：使用统计方法分析数据，验证假设。

**评估指标**：
- 统计方法选择（t检验、卡方检验、ANOVA等）
- 统计显著性（p值<0.05）
- 效应量（效应量大小）

**评分标准**（0-10分）：
- **9-10分**：统计方法正确，统计显著，效应量大
- **7-8分**：统计方法正确，统计显著，效应量中等
- **5-6分**：统计方法正确，统计显著，效应量小
- **3-4分**：统计方法错误，统计不显著
- **0-2分**：统计分析失败，无法验证假设

**案例**：
- **投资分析**：统计分析9/10，统计方法正确
- **产品分析**：统计分析8/10，统计方法正确

---

### 2.5 结论验证（Conclusion Verification）

**定义**：验证实验结论，确保结论可靠。

**评估指标**：
- 结论一致性（结论与假设一致）
- 结论可靠性（结论可重复）
- 结论推广性（结论可推广）

**评分标准**（0-10分）：
- **9-10分**：结论一致、可靠、可推广
- **7-8分**：结论一致、可靠、较可推广
- **5-6分**：结论一致、可靠、推广性一般
- **3-4分**：结论不一致、不可靠、不可推广
- **0-2分**：结论验证失败，结论不可靠

**案例**：
- **投资结论**：结论验证9/10，结论一致、可靠
- **产品结论**：结论验证8/10，结论一致、可靠

## 三、假设测试评分计算

### 3.1 计算公式

```
假设测试总分 = (假设提出 + 实验设计 + 数据收集 + 统计分析 + 结论验证) / 5
```

### 3.2 评分标准

| 总分 | 假设测试等级 | 决策建议 |
|------|-------------|----------|
| **≥8分** | 优秀假设测试 | 可以优先决策 |
| **6-8分** | 良好假设测试 | 可以适度决策 |
| **4-6分** | 一般假设测试 | 谨慎决策 |
| **<4分** | 较差假设测试 | 不建议决策 |

## 四、假设测试验证清单

### 4.1 必须验证的问题

1. [ ] 假设是否清晰可验证？
2. [ ] 实验设计是否科学？
3. [ ] 数据收集是否充足？
4. [ ] 统计分析是否正确？
5. [ ] 结论验证是否可靠？

### 4.2 决策风险警示

- **假设不清晰**：可能导致实验设计错误
- **实验设计不科学**：可能导致数据收集偏差
- **数据收集不足**：可能导致统计分析不准确
- **统计分析错误**：可能导致结论不可靠
- **结论验证不足**：可能导致结论不可推广

## 六、最佳实践

### 6.1 假设测试方法

1. **假设提出**：提出清晰、可验证、可证伪的假设
2. **实验设计**：设计科学实验，控制变量，随机分组
3. **数据收集**：收集高质量、充足、多样的数据
4. **统计分析**：使用正确的统计方法，验证统计显著性
5. **结论验证**：验证结论一致性、可靠性、推广性

### 6.2 常见误区

1. **假设不清晰**：假设应清晰表述
2. **实验设计不科学**：实验设计应科学严谨
3. **数据收集不足**：数据量应充足（≥30个样本）
4. **统计分析错误**：统计方法应正确选择
5. **结论验证不足**：结论应可重复验证

---

## 七、假设测试决策流程

### 7.1 决策步骤

1. **假设提出评估**：评估假设清晰性、可验证性、可证伪性
2. **实验设计评估**：评估实验设计科学性
3. **数据收集评估**：评估数据质量、数据量、数据多样性
4. **统计分析评估**：评估统计方法正确性、统计显著性
5. **结论验证评估**：评估结论一致性、可靠性、推广性
6. **综合评分**：计算假设测试总分
7. **给出决策建议**：ACCEPT/MODIFY/REJECT

### 7.2 决策标准

- **ACCEPT（接受假设）**：总分≥8分，p值<0.05
- **MODIFY（修改假设）**：总分6-8分，p值接近0.05
- **REJECT（拒绝假设）**：总分<6分，p值>0.05
