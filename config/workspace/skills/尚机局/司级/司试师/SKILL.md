## 工作流程

```
接收任务
    ↓
判断自身能力
    ↓
[自身可执行] → 直接执行 → 汇报上级
    ↓
[需要工具] → 调用外部工具 → 执行分析 → 汇报上级
    ↓
[需要协调] → 上报上级 → 等待资源 → 执行任务 → 汇报上级
```

## 常见问题处理

### Q1: 如何判断是否触发子Agent？

**判断标准**：
- 简单实验 → 自身执行
- 复杂模型 → 触发外部工具
- 外部数据 → 触发neodata
- 跨局协调 → 触发上级

### Q2: 如何设计实验？

**步骤**：
1. 明确实验假设
2. 设计实验方案
3. 设置对照组和实验组
4. 执行实验并追踪数据
5. 分析结果并提供建议
