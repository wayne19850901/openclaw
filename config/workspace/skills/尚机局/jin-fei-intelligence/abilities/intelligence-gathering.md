# 情报搜集能力

**归属：** 尚机局·谨妃
**用途：** 搜集金融、科技、媒体、教育四大领域的最新情报

---

## 目标

```
高效搜集四大领域的最新新闻、事件、政策
```

---

## 禁令

```markdown
【禁令】绝对不能在未核实信源前采用情报。

原因：未核实的情报可能是虚假信息。

违反后果：误导决策、信誉损失。
```

```markdown
【禁令】绝对不能只依赖单一信源。

原因：单一信源可能存在偏见或错误。

违反后果：情报片面、判断失误。
```

---

## 四大领域

| 领域 | 范围 | 关键词 |
|------|------|--------|
| **金融** | 贷款方面 | 贷款政策、利率调整、信贷产品、监管政策 |
| **科技** | AI应用方面 | AI技术、大模型、AI应用、行业动态 |
| **媒体** | 自媒体运营 | 平台规则、流量趋势、内容政策、运营方法 |
| **教育** | 应试指南 | 考试政策、考纲变化、招生信息、备考策略 |

---

## 搜集方法

### 方法一：网页搜索

```
使用 web_search 工具
├── 关键词组合搜索
├── 时间范围限定（最近24小时）
└── 来源可信度筛选
```

### 方法二：新闻源订阅

```
可信新闻源列表：
├── 金融：财经网、新浪财经、央行官网
├── 科技：36氪、虎嗅、机器之心
├── 媒体：新榜、卡思数据、巨量算数
└── 教育：教育部官网、各省考试院
```

### 方法三：社交媒体监控

```
监控平台：
├── 微博热搜
├── 微信公众号
├── 抖音热点
└── 小红书趋势
```

## 情报筛选标准

| 重要度 | 标准 | 示例 |
|--------|------|------|
| ⭐⭐⭐ | 重大政策变化、行业重大事件 | 考纲重大调整、AI大模型发布 |
| ⭐⭐ | 行业动态、趋势变化 | 新平台规则、利率调整 |
| ⭐ | 一般性新闻、常规更新 | 常规政策解读、市场动态 |
