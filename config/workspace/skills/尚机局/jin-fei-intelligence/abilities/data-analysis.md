# 数据分析能力

**归属：** 尚机局·谨妃
**用途：** 分析情报数据，提取关键洞察

---

## 目标

```
从情报数据中提取有价值的关键信息和洞察
```

---

## 禁令

```markdown
【禁令】绝对不能编造或推测没有数据支撑的结论。

原因：编造比承认不知道更危险。

违反后果：错误决策、信任危机。
```

---

## 分析框架

### 1. 信息提取

```
原始情报
    ↓
提取关键要素：
├── 人物/机构
├── 事件/变化
├── 时间节点
├── 影响范围
└── 数据指标
```

### 2. 影响评估

```
评估维度：
├── 对威帝王国的影响
│   ├── 金融业务影响
│   ├── 科技发展影响
│   ├── 媒体运营影响
│   └── 教育业务影响
├── 紧迫程度
│   ├── 需立即关注
│   ├── 近期需关注
│   └── 长期观察
└── 行动建议
    ├── 需要决策
    ├── 需要行动
    └── 仅需知晓
```

### 3. 趋势判断

```
趋势类型：
├── 上升趋势 → 机会
├── 下降趋势 → 风险
├── 波动趋势 → 观察
└── 稳定趋势 → 常规
```

---

## 分析维度

| 维度 | 问题 | 输出 |
|------|------|------|
| **WHAT** | 发生了什么？ | 事件描述 |
| **WHO** | 涉及谁？ | 相关方 |
| **WHEN** | 何时发生？ | 时间线 |
| **WHERE** | 在哪里？ | 地点/平台 |
| **WHY** | 为什么重要？ | 影响分析 |
| **HOW** | 如何应对？ | 行动建议 |

---
