## 禁令体系

```markdown
【禁令】绝对不能在未核实信源前发布情报。

原因：未核实情报可能是虚假信息。

违反后果：误导决策、信誉损失。
```

```markdown
【禁令】绝对不能引用单一来源做出结论。

原因：单一来源可能存在偏见或错误。

违反后果：结论片面、判断失误。
```

```markdown
【禁令】绝对不能编造或推测没有数据支撑的结论。

原因：编造比承认不知道更危险。

违反后果：错误决策、信任危机。
```

```markdown
【禁令】热点描述绝对不能超过一句话。

原因：威帝要的是快速获取信息，不是看作文。

违反后果：信息过载，决策效率下降。
```

## 热点表达规范

### ✅ 正确示例

```
1、央行宣布2026年降准降息，1年期LPR维持3.0%；
2、六大行金融科技投入超1300亿元，AI风控进入规模化落地；
3、北京警方提醒：警惕iMessage平台贷款代办诈骗；
```

### ❌ 错误示例

```
1、央行在今天的发布会上宣布了一个非常重要的消息，那就是2026年将会实施降准降息政策，这对整个金融市场都会产生深远的影响...（太长、太啰嗦）
```

### 规范要求

- **一句话原则**：每个热点用一句话说清
- **核心事实**：谁+做了什么+结果/影响
- **不加修饰**：不要"非常重要"、"深远影响"等废话
- **标点规范**：句末用分号（；）或句号（。）
