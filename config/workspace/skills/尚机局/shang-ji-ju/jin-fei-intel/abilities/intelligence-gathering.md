# 情报搜集

## 概述

搜集市场情报、新闻资讯、竞争对手信息，为威帝提供决策支持。

## 搜集领域

威帝王国四大领域：

| 领域 | 关注点 |
|------|--------|
| **金融** | 贷款政策、利率变化、监管动态 |
| **科技** | AI应用、新技术、行业趋势 |
| **媒体** | 自媒体运营、平台政策、内容趋势 |
| **教育** | 应试指南、教育政策、培训市场 |

## 搜集渠道

### 公开渠道
- 新闻网站与行业媒体
- 政府公告与政策文件
- 社交媒体舆情
- 企业公开报告

### 技术工具
- 网页抓取：`url-to-markdown`
- 社交监测：`social-monitoring`
- 技能发现：`find-skills`

## 搜集流程

```
确定情报主题
    ↓
选择搜集渠道
    ↓
执行信息抓取
    ↓
筛选与去重
    ↓
验证信息真实性
    ↓
结构化整理
    ↓
提交分析
```

## 信息筛选标准

| 优先级 | 标准 |
|--------|------|
| 🔴 高 | 直接影响威帝王国业务 |
| 🟡 中 | 行业重要动态，间接影响 |
| 🟢 低 | 一般性资讯，参考价值 |

## 禁令

| 禁令 | 原因 |
|------|------|
| 禁止编造情报 | 真实性是情报的生命 |
| 禁止传播未经证实消息 | 可信度红线 |
| 禁止泄露搜集来源 | 安全红线 |
| 禁止非法获取信息 | 法律红线 |
