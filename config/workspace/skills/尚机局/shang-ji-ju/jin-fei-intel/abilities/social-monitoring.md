# 社交媒体监测

## 概述

监测X/Twitter、微博等社交媒体内容，用于舆情分析。

## 使用方式

```bash
/baoyu-danger-x-to-markdown https://twitter.com/user/status/xxx
```

## 监测维度

| 维度 | 指标 |
|------|------|
| 热度 | 点赞、转发、评论 |
| 情感 | 正面/负面/中性 |
| 趋势 | 话题热度变化 |
| 影响力 | 发言者影响力 |

## 输出格式

```markdown
# 社交媒体监测报告

**平台**：{platform}
**时间范围**：{start} - {end}
**关键词**：{keywords}

## 热门内容

### 1. {标题}
- **作者**：{author}
- **互动量**：{interactions}
- **链接**：{url}
- **摘要**：{summary}

### 2. ...

## 情感分析

| 情感 | 比例 |
|------|------|
| 正面 | {pos}% |
| 中性 | {neu}% |
| 负面 | {neg}% |

## 趋势洞察

{insights}
```
