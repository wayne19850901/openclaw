# 知识管理

## 概述

负责情报资产的沉淀，管理外部数据供应商，确保采集动作合规。

## 核心职责

- 知识图谱建设
- 数据供应商管理
- 合规对接
- 资产沉淀

## 知识图谱

### 实体类型

| 实体 | 示例 |
|------|------|
| 公司 | 某某教育、某某科技 |
| 人物 | 创始人、高管 |
| 产品 | App、课程 |
| 事件 | 融资、并购 |
| 政策 | 法规、文件 |

### 关系类型

| 关系 | 示例 |
|------|------|
| 投资 | A公司投资B公司 |
| 竞争 | A公司与B公司竞争 |
| 合作 | A公司与B公司合作 |
| 任职 | 某人任职某公司 |

## 供应商管理

| 供应商 | 类型 | 内容 |
|--------|------|------|
| 万得 | 金融数据 | 行情、公告 |
| 彭博 | 全球财经 | 新闻、数据 |
| 企查查 | 企业信息 | 工商、法律 |

## 合规对接

- 与法务部门确认采集红线
- 定期审计采集行为
- 处理合规风险预警
