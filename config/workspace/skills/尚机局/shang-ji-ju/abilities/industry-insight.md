# 行业洞察

## 概述

研究竞品动态、用户行为变迁，输出具有实操价值的行业内参。

## 核心职责

- 行业政策解读
- 用户行为研究
- 市场趋势捕捉
- 内参报告输出

## 研究框架

```
┌─────────────────────────────────────────────────────────┐
│ 政策层                                                  │
│    ├── 法规变化                                         │
│    ├── 监管动向                                         │
│    └── 标准制定                                         │
├─────────────────────────────────────────────────────────┤
│ 市场层                                                  │
│    ├── 市场规模                                         │
│    ├── 增长趋势                                         │
│    └── 细分机会                                         │
├─────────────────────────────────────────────────────────┤
│ 用户层                                                  │
│    ├── 用户画像                                         │
│    ├── 行为变迁                                         │
│    └── 需求演化                                         │
├─────────────────────────────────────────────────────────┤
│ 竞争层                                                  │
│    ├── 竞品动态                                         │
│    ├── 新玩家                                           │
│    └── 替代威胁                                         │
└─────────────────────────────────────────────────────────┘
```

## 行业覆盖

| 行业 | 重点方向 |
|------|----------|
| 教育 | 中考、高考、职业培训 |
| 金融 | 贷款、理财、保险 |
| 科技 | AI应用、SaaS |
| 媒体 | 自媒体、内容平台 |
