# 竞争对手情报

## 概述

针对重点竞争对手进行长期跟踪，预判其下一步动作。

## 核心职责

- 竞品信息收集
- 战略路径还原
- 动向预判
- 预警报告

## 跟踪框架

```
┌─────────────────────────────────────────────────────────┐
│ 信息层                                                  │
│    ├── 产品动态                                         │
│    ├── 人事变动                                         │
│    ├── 融资信息                                         │
│    ├── 市场活动                                         │
│    └── 技术进展                                         │
├─────────────────────────────────────────────────────────┤
│ 分析层                                                  │
│    ├── 战略意图解读                                     │
│    ├── 能力边界评估                                     │
│    ├── 弱点识别                                         │
│    └── 路径还原                                         │
├─────────────────────────────────────────────────────────┤
│ 预判层                                                  │
│    ├── 下一步动作预测                                   │
│    ├── 攻击可能性评估                                   │
│    └── 应对策略建议                                     │
└─────────────────────────────────────────────────────────┘
```

## 信息来源

| 来源 | 类型 | 信息量 |
|------|------|--------|
| 公开信息 | 官网、公告、新闻 | 80% |
| 社交媒体 | 微博、公众号、知乎 | 10% |
| 招聘信息 | 职位、薪资、要求 | 5% |
| 专利/论文 | 技术方向 | 3% |
| 用户反馈 | App评分、评论 | 2% |

## 产出物

| 类型 | 频率 | 内容 |
|------|------|------|
| 竞品周报 | 每周 | 动态汇总 |
| 专题分析 | 按需 | 深度研究 |
| 预警简报 | 实时 | 重要变化 |
