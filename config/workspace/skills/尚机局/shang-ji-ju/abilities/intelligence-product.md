## 报告自动化

```python
def generate_daily_report():
    # 1. 聚合当日情报
    intel = aggregate_intel(date=today)
    
    # 2. AI生成摘要
    summary = ai_summarize(intel)
    
    # 3. 排序和分类
    categorized = categorize_and_rank(intel)
    
    # 4. 生成报告
    report = render_template(categorized, summary)
    
    # 5. 分发
    distribute(report, channels=['email', 'wechat'])
```
