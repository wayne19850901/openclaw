# 金融宏观分析

## 概述

穿透分析金融动向及其对教育、软件等业务的联动机遇。

## 核心职责

- 金融动向分析
- 宏观经济解读
- 资本市场跟踪
- 跨行业联动分析

## 分析框架

```
┌─────────────────────────────────────────────────────────┐
│ 宏观层面                                                │
│    ├── GDP、CPI、PMI等宏观指标                          │
│    ├── 货币政策（利率、流动性）                         │
│    └── 财政政策（税收、支出）                           │
├─────────────────────────────────────────────────────────┤
│ 市场层面                                                │
│    ├── 股市动态                                         │
│    ├── 债市走势                                         │
│    ├── 汇率变化                                         │
│    └── 大宗商品                                         │
├─────────────────────────────────────────────────────────┤
│ 行业层面                                                │
│    ├── 教育行业资本动态                                 │
│    ├── 科技行业融资趋势                                 │
│    └── 跨行业并购机会                                   │
├─────────────────────────────────────────────────────────┤
│ 公司层面                                                │
│    ├── 竞争对手融资                                     │
│    ├── 合作伙伴动态                                     │
│    └── 供应链风险                                       │
└─────────────────────────────────────────────────────────┘
```

## 产出物

| 类型 | 频率 | 内容 |
|------|------|------|
| 金融日报 | 每日 | 市场动态摘要 |
| 周度分析 | 每周 | 趋势解读 |
| 专题报告 | 按需 | 深度研究 |
| 预警简报 | 实时 | 风险提示 |

## 分析维度

| 维度 | 关注点 |
|------|--------|
| 利率变化 | 对融资成本的影响 |
| 政策调整 | 对行业格局的影响 |
| 资本流向 | 对竞争态势的影响 |
| 汇率波动 | 对跨境业务的影响 |
