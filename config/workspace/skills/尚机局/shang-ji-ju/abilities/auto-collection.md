# 自动化采集

## 概述

负责信息的获取与自动化，建立分布式爬虫系统和RPA流水线。

## 核心职责

- 分布式爬虫系统开发
- RPA自动化流水线搭建
- 反爬对抗策略
- 多源数据采集

## 采集架构

```
┌─────────────────────────────────────────────────────────┐
│ 数据源层                                                │
│    ├── 金融行情（万得、同花顺）                          │
│    ├── 媒体舆情（微博、公众号、新闻）                    │
│    ├── 教育平台（考试院、学校官网）                      │
│    └── 公开数据（政府公开、企业信息）                    │
├─────────────────────────────────────────────────────────┤
│ 采集层                                                  │
│    ├── 分布式爬虫集群                                   │
│    ├── RPA自动化                                        │
│    └── API对接                                          │
├─────────────────────────────────────────────────────────┤
│ 处理层                                                  │
│    ├── 反爬对抗                                         │
│    ├── 数据清洗                                         │
│    └── 格式标准化                                       │
├─────────────────────────────────────────────────────────┤
│ 存储层                                                  │
│    └── 情报数据库                                       │
└─────────────────────────────────────────────────────────┘
```

## 采集目标

| 领域 | 数据源 | 频率 |
|------|--------|------|
| 金融 | 行情、公告、研报 | 实时/日更 |
| 科技 | 技术博客、开源动态 | 日更 |
| 媒体 | 微博、公众号、新闻 | 小时级 |
| 教育 | 考试政策、招生信息 | 日更 |

## 工具栈

| 工具 | 用途 |
|------|------|
| Scrapy | 分布式爬虫 |
| 影刀ShadowBot | RPA自动化 |
| Playwright | 动态渲染 |
| Apache Kafka | 消息队列 |
