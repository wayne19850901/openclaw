# 情报Dashboard设计

## 设计原则

### 信息层级

```
第一眼 → 核心指标、预警信息
第二眼 → 趋势图表、关键动态
深入看 → 详细列表、历史数据
```

### 视觉设计

| 原则 | 说明 |
|------|------|
| 重点突出 | 重要信息用颜色/大小强调 |
| 一致性 | 统一的配色和字体 |
| 留白 | 避免信息过载 |
| 响应式 | 适配多种屏幕 |

## 模块设计

### 顶部导航

- 快速搜索
- 用户设置
- 预警通知入口

### 首页布局

```
┌─────────────────────────────────────────────────┐
│ 📊 核心指标卡片                                   │
│    [情报量] [预警数] [处理率] [准确率]            │
├─────────────────────────────────────────────────┤
│ 🔥 热点情报流                                     │
│    实时滚动，支持筛选和搜索                       │
├─────────────────────────────────────────────────┤
│ 📈 趋势图表                                       │
│    时间序列、分类分布、情感分析                   │
├─────────────────────────────────────────────────┤
│ ⚠️ 预警面板                                       │
│    重要事项、紧急提醒                             │
└─────────────────────────────────────────────────┘
```

## 交互设计

### 筛选器

- 时间范围
- 领域分类
- 重要度
- 来源类型

### 下钻操作

- 点击查看详情
- 关联情报推荐
- 导出报告

## 移动端适配

| 场景 | 设计 |
|------|------|
| 快速浏览 | 卡片式布局 |
| 推送通知 | 重要信息直达 |
| 离线查看 | 本地缓存 |
