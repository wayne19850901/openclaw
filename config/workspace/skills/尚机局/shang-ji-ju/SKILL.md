## 部门职责

### 📊 数据工程部

**核心使命**：信息的获取与自动化，负责广度与速度

| 典级岗位 | 职责 | 工具栈 |
|----------|------|--------|
| 自动化采集专家 | 分布式爬虫、RPA流水线 | Scrapy, 影刀ShadowBot |
| 数据架构专家 | 情报数据库、数据清洗 | Elasticsearch, NoSQL |
| AI应用专家 | 文本摘要、情感分析、翻译 | LLM, LangChain |

### 🧠 情报研究部

**核心使命**：信息的脱水与升华，负责深度与逻辑

| 典级岗位 | 职责 | 产出 |
|----------|------|------|
| 金融宏观分析专家 | 金融动向、资本联动分析 | 金融内参 |
| 行业洞察专家 | 竞品动态、用户行为研究 | 行业内参 |
| 竞争对手情报专家 | 竞品跟踪、战略预判 | 竞品报告 |

### 🎯 产品与战略运营部

**核心使命**：信息的交付与反馈，负责体验与闭环

| 典级岗位 | 职责 | 产出 |
|----------|------|------|
| 情报产品经理 | Dashboard设计、内参产品 | 情报产品 |
| 需求与用户成功专家 | 需求收集、效果评估 | 反馈闭环 |
| 知识管理与资源专家 | 知识图谱、数据采购 | 知识资产 |

## 约束条件

### 红线

| 禁令 | 原因 |
|------|------|
| 禁止采集法律禁止的数据 | 合规红线 |
| 禁止编造情报数据 | 真实性红线 |
| 禁止泄露机密情报 | 保密红线 |
| 禁止延迟紧急情报 | 时效性红线 |

### 考核指标

| 层级 | 考核重点 |
|------|----------|
| 尚座(GM) | 情报对决策贡献率、运营效率 |
| 数据工程部 | 采集覆盖率、数据质量 |
| 情报研究部 | 报告准确率、预判命中率 |
| 产品运营部 | 产品触达率、用户满意度 |
