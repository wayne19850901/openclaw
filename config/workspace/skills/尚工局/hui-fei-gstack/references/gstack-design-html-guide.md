# gstack Design-HTML 完整指南

## 来源

整合自 `~/.claw/workspace/skills/gstack/design-html/SKILL.md`

## 核心概念

gstack design-html 使用Pretext文本布局引擎生成生产级HTML，确保精确的高度计算和布局控制。

## Pretext布局引擎

Pretext是一个文本布局引擎，专门解决HTML中高度计算不准确的问题：
- 精确的高度计算
- 正确的文本截断
- 响应式布局适配
- 打印样式支持

## 智能API路由

根据设计类型自动选择最优生成路径：

| 设计类型 | 路由策略 |
|----------|----------|
| 文本密集 | Pretext精确排版 |
| 交互密集 | 组件化生成 |
| 动画密集 | CSS动画优先 |
| 数据驱动 | 模板化生成 |

## 四种Pretext布线模式

### 1. 单页应用（SPA）
- 客户端路由
- 状态管理
- 懒加载
- 适用于交互密集型应用

### 2. 多页站点（MPA）
- 服务端渲染
- SEO友好
- 渐进增强
- 适用于内容型网站

### 3. 组件库
- 独立组件
- 文档自动生成
- 版本管理
- 适用于设计系统

### 4. 落地页
- 高度精确
- 动画流畅
- 转化优化
- 适用于营销页面
