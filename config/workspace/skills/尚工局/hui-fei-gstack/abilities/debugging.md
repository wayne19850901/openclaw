# 问题排查能力

> 尚工局慧妃原有能力

## 概述

系统化排查问题，定位根因，制定修复方案。

## 执行流程

1. **问题确认**
   - 复现问题
   - 收集日志
   - 确认影响范围

2. **问题分析**
   - 时间线梳理
   - 相关变更排查
   - 依赖服务检查

3. **根因定位**
   - 日志分析
   - 代码审查
   - 环境对比

4. **修复方案**
   - 制定修复计划
   - 评估风险
   - 实施修复

5. **预防措施**
   - 添加监控
   - 补充测试
   - 更新文档

## 排查方法

| 方法 | 适用场景 |
|------|----------|
| 日志分析 | 运行时错误 |
| 代码审查 | 逻辑错误 |
| 环境对比 | 配置问题 |
| 性能分析 | 性能问题 |
| 网络抓包 | 网络问题 |

## 输出格式

```
## 问题排查报告

### 问题描述
- 现象：...
- 影响：...
- 时间：...

### 根因分析
- 直接原因：...
- 根本原因：...
- 触发条件：...

### 修复方案
- 临时修复：...
- 长期修复：...

### 预防措施
- 监控：...
- 测试：...
- 文档：...
```
