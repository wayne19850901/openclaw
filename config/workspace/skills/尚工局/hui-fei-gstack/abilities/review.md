# 代码审查能力

> 整合自 gstack/review — Pre-landing PR审查工作流

## 概述

执行系统化的代码审查，覆盖安全、性能、正确性等关键维度。

## 触发条件

- PR提交
- 代码变更合并前
- 用户明确要求审查

## 执行流程

1. **加载完整指令**：读取 `~/.claw/workspace/skills/gstack/review/SKILL.md`
2. **分类审查**：
   - SQL安全：注入检测、事务隔离级别、连接池泄漏
   - 竞态条件：锁粒度、原子操作、TOCTOU
   - LLM信任边界：输入验证、输出转义、prompt注入
   - Shell注入：命令构造、参数转义
3. **专家派遣**：
   - testing专家：测试覆盖、边界条件
   - security专家：安全漏洞、权限检查
   - performance专家：性能瓶颈、资源泄漏
4. **对抗审查**：Claude/Codex交叉验证
5. **输出审查报告**

## 关键审查类别

| 类别 | 检查项 |
|------|--------|
| SQL安全 | 注入、事务隔离、连接泄漏 |
| 竞态条件 | 锁粒度、原子操作、TOCTOU |
| LLM信任 | 输入验证、输出转义、prompt注入 |
| Shell注入 | 命令构造、参数转义 |
| 错误处理 | 异常捕获、错误传播、降级策略 |
| 资源管理 | 内存泄漏、文件句柄、连接池 |

## 输出格式

```
## 审查报告

### 关键问题（必须修复）
- [问题] 描述 → 建议

### 建议改进（推荐修复）
- [建议] 描述 → 建议

### 肯定（做得好的地方）
- [肯定] 描述

### 审查结论
APPROVE / REQUEST_CHANGES / NEEDS_DISCUSSION
```
