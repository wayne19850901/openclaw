# HTML生成能力

> 整合自 gstack/design-html — Pretext原生HTML生成

## 概述

使用Pretext文本布局引擎生成生产级HTML，支持智能API路由和实时重载。

## 触发条件

- 设计定稿需要生产HTML
- 需要精确高度计算的布局
- 用户明确要求HTML生成

## 执行流程

1. **加载完整指令**：读取 `~/.claw/workspace/skills/gstack/design-html/SKILL.md`
2. **智能API路由**：
   - 分析设计类型
   - 选择最优生成路径
3. **选择Pretext布线模式**：
   - 单页应用（SPA）
   - 多页站点（MPA）
   - 组件库
   - 落地页
4. **生成Pretext原生HTML**：
   - 正确高度计算
   - 响应式布局
   - 语义化标签
5. **启动实时重载服务器**：
   - 文件变更自动刷新
   - 快速迭代验证
6. **迭代精化**：
   - 基于预览调整
   - 微调样式和布局
7. **输出最终HTML**

## Pretext布线模式

| 模式 | 适用场景 | 特点 |
|------|----------|------|
| SPA | 交互密集型应用 | 客户端路由、状态管理 |
| MPA | 内容型网站 | SEO友好、渐进增强 |
| 组件库 | 设计系统 | 独立组件、文档生成 |
| 落地页 | 营销页面 | 高度精确、动画流畅 |

## 智能API路由

根据设计特征自动选择：
- 文本密度高 → Pretext精确排版
- 交互复杂 → 组件化生成
- 动画密集 → CSS动画优先
- 数据驱动 → 模板化生成
