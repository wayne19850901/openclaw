# 代码开发能力

> 尚工局慧妃原有能力

## 概述

编写高质量代码，遵循TDD原则，确保代码可维护、可测试。

## 执行流程

1. **理解需求**
   - 分析功能需求
   - 理解技术约束
   - 确认验收标准

2. **设计接口**
   - 定义模块接口
   - 设计数据结构
   - 规划错误处理

3. **编写测试**
   - 单元测试先行
   - 覆盖边界条件
   - 集成测试补充

4. **实现功能**
   - 遵循编码规范
   - 保持代码简洁
   - 添加必要注释

5. **代码审查**
   - 自查代码质量
   - 提交审查
   - 处理审查意见

## 编码规范

- 命名：语义化、一致性
- 函数：单一职责、短小精悍
- 注释：解释为什么，不是做什么
- 错误处理：明确、可恢复
