# RAG与提示词工程

## 概述

提升AI的垂直专业度，负责向量数据库搭建、复杂Prompt链条的编写与测试。

## 核心职责

- 向量数据库设计与优化
- RAG效果调优
- Prompt链条编写
- 抗攻击提示词设计

## RAG架构

```
┌─────────────────────────────────────────────────────────┐
│ 用户查询                                                 │
│    ↓                                                    │
│ Embedding（向量化）                                       │
│    ↓                                                    │
│ 向量检索（相似度搜索）                                     │
│    ↓                                                    │
│ Rerank（重排序）                                          │
│    ↓                                                    │
│ Context组装                                              │
│    ↓                                                    │
│ LLM生成                                                  │
│    ↓                                                    │
│ 输出答案                                                 │
└─────────────────────────────────────────────────────────┘
```

## Prompt工程

### Prompt模板结构

```markdown
## 角色
{定义AI的角色和能力边界}

## 任务
{明确任务目标}

## 上下文
{提供必要的背景信息}

## 约束
{列出限制条件}

## 输出格式
{定义输出结构}

## 示例
{提供few-shot示例}
```

### 抗攻击策略

| 攻击类型 | 防护策略 |
|----------|----------|
| Prompt注入 | 输入过滤、角色隔离 |
| 越权访问 | 权限检查、上下文隔离 |
| 数据泄露 | 输出过滤、敏感词检测 |
