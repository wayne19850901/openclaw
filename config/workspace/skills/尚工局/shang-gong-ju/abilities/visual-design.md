# 视觉设计

## 概述

维护高标准的视觉一致性，利用AI工具大规模产出品牌素材、图标及应用内插画。

## 核心职责

- 品牌视觉系统维护
- AI素材大规模生成
- 视觉一致性保障
- 专有模型训练

## 视觉系统

### 品牌调性

| 元素 | 定义 |
|------|------|
| 主色 | 品牌主色调 |
| 辅色 | 辅助配色系统 |
| 字体 | 品牌字体规范 |
| 图标 | 图标设计系统 |
| 插画 | 插画风格指南 |

### AI素材生成

```
需求 → 风格定义 → Prompt编写 → AI生成 → 筛选优化 → 输出
```

## 工具栈

| 工具 | 用途 |
|------|------|
| Stable Diffusion | 图像生成 |
| Midjourney | 创意图像 |
| Adobe Firefly | 商业素材 |
| Flux | 高质量生成 |
| ComfyUI | 工作流自动化 |
| Tailwind CSS | 样式系统 |

## 专有模型训练

```bash
# 品牌风格LoRA训练
1. 收集品牌素材（50-100张）
2. 标注风格特征
3. 训练LoRA模型
4. 测试与调优
5. 部署使用
```

## 输出物

- 品牌视觉规范手册
- 图标库
- 插画库
- AI模型（LoRA/Style）
