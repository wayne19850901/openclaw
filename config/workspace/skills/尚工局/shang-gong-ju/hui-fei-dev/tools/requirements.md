# 工具依赖清单

## 核心工具

| 工具 | 版本要求 | 安装命令 |
|------|----------|----------|
| Node.js | 18+ | `brew install node` 或 `nvm install 18` |
| Bun | latest | `brew install oven-sh/bun/bun` |
| Git | 2.x | `brew install git` |

## 尚科局工具

| 工具 | 用途 | 安装命令 |
|------|------|----------|
| Vercel CLI | 部署 | `npm install -g vercel` |
| React Native CLI | 移动开发 | `npm install -g react-native-cli` |

## 尚宣局工具

| 工具 | 用途 | 说明 |
|------|------|------|
| Chrome | 公众号发布 | 浏览器方式必需 |
| 图片生成API | 内容创作 | 需配置API密钥 |

## 尚宫局工具

| 工具 | 用途 | 安装命令 |
|------|------|----------|
| GitHub CLI | 仓库管理 | `brew install gh` |

## 环境变量

```bash
# 微信公众号（尚宣局/尚教局）
WECHAT_APP_ID=your_app_id
WECHAT_APP_SECRET=your_app_secret

# Vercel（尚科局）
VERCEL_TOKEN=your_token

# GitHub（尚宫局）
GH_TOKEN=your_token
```
