## 一、价值投资定义

**价值投资（Value Investing）**：Ben Graham 提出的投资方法，通过购买价格低于内在价值的股票，获得安全边际保护。

**核心原则**：
- 价格与价值分离（价格围绕价值波动）
- 安全边际保护（买入价格≤内在价值×70%）
- 市场先生理论（市场情绪波动提供机会）
- 长期持有策略（时间是价值投资的朋友）

## 二、价值投资要素

### 2.1 内在价值计算（Intrinsic Value Calculation）

**定义**：计算企业的内在价值，作为投资决策的基准。

**评估指标**：
- 资产价值（净资产、流动资产）
- 盈利能力价值（净利润、自由现金流）
- 成长价值（增长率、成长期限）

**计算方法**：
```
内在价值 = 资产价值 + 盈利能力价值 + 成长价值
```

**评分标准**（0-10分）：
- **9-10分**：内在价值计算准确，误差<10%
- **7-8分**：内在价值计算较准确，误差10-20%
- **5-6分**：内在价值计算一般，误差20-30%
- **3-4分**：内在价值计算较差，误差>30%
- **0-2分**：内在价值计算失败，无法计算

**案例**：
- **贵州茅台**：内在价值计算9/10，计算准确
- **中国平安**：内在价值计算8/10，计算较准确

---

### 2.2 安全边际评估（Margin of Safety Assessment）

**定义**：评估买入价格与内在价值之间的安全边际。

**评估指标**：
- 安全边际比例（安全边际≥30%）
- 安全边际稳定性（安全边际波动性）
- 安全边际保护（安全边际对风险的保护）

**计算公式**：
```
安全边际 = (内在价值 - 买入价格) / 内在价值 × 100%
```

**评分标准**（0-10分）：
- **9-10分**：安全边际≥50%，保护极强
- **7-8分**：安全边际30-50%，保护较强
- **5-6分**：安全边际20-30%，保护一般
- **3-4分**：安全边际10-20%，保护较弱
- **0-2分**：安全边际<10%，无保护

**案例**：
- **贵州茅台**：安全边际9/10，安全边际≥50%
- **中国平安**：安全边际8/10，安全边际30-50%

### 2.3 格雷厄姆数字（Graham Number）

**定义**：Ben Graham 提出的快速估值方法，用于筛选价值股。

**计算公式**：
```
格雷厄姆数字 = √(22.5 × 每股收益 × 每股净资产)
```

**评估标准**：
- **买入信号**：股价<格雷厄姆数字×70%
- **持有信号**：股价在格雷厄姆数字×70%-100%
- **卖出信号**：股价>格雷厄姆数字×100%

**评分标准**（0-10分）：
- **9-10分**：格雷厄姆数字计算准确，筛选效果显著
- **7-8分**：格雷厄姆数字计算较准确，筛选效果良好
- **5-6分**：格雷厄姆数字计算一般，筛选效果一般
- **3-4分**：格雷厄姆数字计算较差，筛选效果较差
- **0-2分**：格雷厄姆数字计算失败，无法筛选

**案例**：
- **贵州茅台**：格雷厄姆数字9/10，计算准确
- **中国平安**：格雷厄姆数字8/10，计算较准确

### 2.4 净净值法（Net-Net Method）

**定义**：Ben Graham 提出的极端价值投资方法，用于筛选深度价值股。

**计算公式**：
```
净净值 = 流动资产 - 总负债 - 优先股
净净值每股价值 = 净净值 / 普通股股数
```

**评估标准**：
- **买入信号**：股价<净净值每股价值×67%
- **持有信号**：股价在净净值每股价值×67%-100%
- **卖出信号**：股价>净净值每股价值×100%

**评分标准**（0-10分）：
- **9-10分**：净净值法计算准确，筛选效果显著
- **7-8分**：净净值法计算较准确，筛选效果良好
- **5-6分**：净净值法计算一般，筛选效果一般
- **3-4分**：净净值法计算较差，筛选效果较差
- **0-2分**：净净值法计算失败，无法筛选

**案例**：
- **深度价值股**：净净值法9/10，计算准确
- **困境反转股**：净净值法8/10，计算较准确

### 2.5 防御型投资组合（Defensive Portfolio）

**定义**：Ben Graham 提出的防御型投资组合策略，用于降低投资风险。

**组合要求**：
- 股票数量≥30只（分散风险）
- 行业分布≥10个（分散行业风险）
- 大型企业（市值≥100亿）
- 强劲财务状况（流动比率≥2，负债权益比<1）

**评分标准**（0-10分）：
- **9-10分**：组合符合所有要求，防御性极强
- **7-8分**：组合符合大部分要求，防御性较强
- **5-6分**：组合符合部分要求，防御性一般
- **3-4分**：组合符合少量要求，防御性较弱
- **0-2分**：组合不符合要求，防御性失败

**案例**：
- **防御型组合**：防御型投资组合9/10，防御性极强
- **平衡型组合**：防御型投资组合8/10，防御性较强

## 三、价值投资评分计算

### 3.1 计算公式

```
价值投资总分 = (内在价值计算 + 安全边际评估 + 格雷厄姆数字 + 净净值法 + 防御型投资组合) / 5
```

### 3.2 评分标准

| 总分 | 价值投资等级 | 投资建议 |

## 六、最佳实践

### 6.1 价值投资方法

1. **内在价值计算**：准确计算企业内在价值
2. **安全边际评估**：评估买入价格与内在价值之间的安全边际
3. **格雷厄姆数字**：使用格雷厄姆数字快速筛选价值股
4. **净净值法**：使用净净值法筛选深度价值股
5. **防御型投资组合**：构建防御型投资组合降低风险

### 6.2 常见误区

1. **忽视内在价值计算**：内在价值是价值投资的基石
2. **安全边际不足**：安全边际是价值投资的保护
3. **过度依赖格雷厄姆数字**：格雷厄姆数字是快速筛选方法
4. **忽视净净值法**：净净值法是深度价值投资方法
5. **投资组合防御性不足**：防御型投资组合降低风险

## 七、价值投资决策流程

### 7.1 决策步骤

1. **内在价值计算评估**：评估内在价值计算准确性
2. **安全边际评估**：评估安全边际充分性
3. **格雷厄姆数字计算**：计算格雷厄姆数字
4. **净净值法计算**：计算净净值每股价值
5. **防御型投资组合评估**：评估投资组合防御性
6. **综合评分**：计算价值投资总分
7. **给出投资建议**：BUY/HOLD/SELL

### 7.2 投资决策标准

- **BUY（买入）**：总分≥8分，安全边际≥30%
- **HOLD（持有）**：总分6-8分，安全边际20-30%
- **SELL（卖出）**：总分<6分，安全边际<20%
