## 工作流概述

本工作流基于 Ben Graham 的价值投资哲学，采用 **内在价值计算 + 安全边际评估 + 格雷厄姆数字 + 净净值法 + 防御型投资组合** 五大核心方法论。

**执行原则**：
1. **强制读完机制**：AI必须执行完整流程，不得跳过任何步骤
2. **数据源唯一性**：所有金融数据必须使用 `neodata-financial-search` skill
3. **安全边际标准**：买入价格≤内在价值×70%，安全边际≥30%
4. **防御型组合**：股票数量≥30只，行业分布≥10个

## Phase 2: 内在价值计算（强制执行）

### 2.1 资产价值计算

**任务**: 计算企业的资产价值。

**计算公式**：
```
资产价值 = 净资产（最新）
资产价值每股 = 资产价值 / 普通股股数
```

**输出格式**：
```
【资产价值计算】
- 最新净资产：[净资产]亿元
- 普通股股数：[股数]亿股
- 资产价值每股：[资产价值每股]元
```

**强制检查点**：必须完成资产价值计算，缺失不得进入下一阶段。

---

### 2.2 盈利能力价值计算

**任务**: 计算企业的盈利能力价值。

**计算公式**：
```
盈利能力价值 = 过去5年平均净利润 / 资本成本率（10%）
盈利能力价值每股 = 盈利能力价值 / 普通股股数
```

**输出格式**：
```
【盈利能力价值计算】
- 过去5年平均净利润：[平均净利润]亿元
- 资本成本率：10%
- 盈利能力价值：[盈利能力价值]亿元
- 盈利能力价值每股：[盈利能力价值每股]元
```

**强制检查点**：必须完成盈利能力价值计算，缺失不得进入下一阶段。

---

### 2.3 内在价值汇总

**任务**: 汇总资产价值和盈利能力价值，计算内在价值。

**计算公式**：
```
内在价值 = (资产价值 + 盈利能力价值) / 2
内在价值每股 = 内在价值 / 普通股股数
```

**输出格式**：
```
【内在价值汇总】
- 资产价值每股：[资产价值每股]元
- 盈利能力价值每股：[盈利能力价值每股]元
- 内在价值每股：[内在价值每股]元
```

**强制检查点**：必须完成内在价值计算，缺失不得进入下一阶段。

## Phase 3: 安全边际评估（强制执行）

### 3.1 安全边际计算

**任务**: 计算安全边际（买入价格上限）。

**安全边际标准**：
- **买入价格 ≤ 内在价值 × 70%**（Ben Graham标准）
- 安全边际 ≥ 30%

**计算公式**：
```
买入价格上限 = 内在价值每股 × 70%
安全边际 = (内在价值每股 - 当前股价) / 内在价值每股 × 100%
```

**输出格式**：
```
【安全边际评估】
- 内在价值每股：[内在价值每股]元
- 买入价格上限（内在价值×70%）：[上限]元/股
- 当前股价：[股价]元/股
- 安全边际：[安全边际]%
```

**强制检查点**：
- 安全边际 ≥ 30%：进入下一阶段
- 安全边际 < 30%：**立即终止，输出"不买入"决策**

## Phase 4: 格雷厄姆数字计算（强制执行）

### 4.1 格雷厄姆数字计算

**任务**: 使用格雷厄姆数字快速筛选价值股。

**计算公式**：
```
格雷厄姆数字 = √(22.5 × EPS × BVPS)
```

**评估标准**：
- **买入信号**：股价 < 格雷厄姆数字 × 70%
- **持有信号**：股价在格雷厄姆数字 × 70%-100%
- **卖出信号**：股价 > 格雷厄姆数字 × 100%

**输出格式**：

## Phase 5: 净净值法计算（强制执行）

### 5.1 净净值计算

**任务**: 使用净净值法筛选深度价值股。

**计算公式**：
```
净净值 = 流动资产 - 总负债 - 优先股
净净值每股价值 = 净净值 / 普通股股数
```

**评估标准**：
- **买入信号**：股价 < 净净值每股价值 × 67%
- **持有信号**：股价在净净值每股价值 × 67%-100%
- **卖出信号**：股价 > 净净值每股价值 × 100%

**输出格式**：
```
【净净值法计算】
- 流动资产：[流动资产]亿元
- 总负债：[总负债]亿元
- 优先股：[优先股]亿元
- 净净值：[净净值]亿元
- 净净值每股价值：[净净值每股价值]元
- 当前股价：[股价]元
- 信号：BUY / HOLD / SELL
```

**强制检查点**：必须完成净净值法计算，缺失不得进入下一阶段。
