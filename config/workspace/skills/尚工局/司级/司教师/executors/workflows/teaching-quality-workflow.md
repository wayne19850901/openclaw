> **约束**: 必须完整执行，不得跳过任何步骤

## 工作流概述

本工作流基于 Charlie Munger 的检查清单方法论，采用 **检查清单验证 + 反向思考 + 多学科思维模型** 三大核心方法论。

**执行原则**：
1. **强制读完机制**：AI必须执行完整流程，不得跳过任何步骤
2. **数据源唯一性**：所有金融数据必须使用 `neodata-financial-search` skill
3. **量化指标验证**：每个决策必须有量化指标支撑（检查清单通过率≥80%）
4. **反向思考验证**：必须进行反向思考，验证失败场景

## Phase 1: 教学数据收集（强制执行）

### 1.1 获取目标公司教学相关数据

**任务**: 使用 `neodata-financial-search` skill 获取目标公司的教学相关数据。

**执行步骤**：
1. 调用 `neodata-financial-search` skill
2. 搜索目标公司的教学数据：
   - 培训投入
   - 培训覆盖率
   - 员工技能提升率
   - 知识资产价值
   - 培训ROI
   - 员工满意度

**输出格式**：
```
【教学数据】
- 培训投入：[投入]亿元
- 培训覆盖率：[覆盖率]%员工
- 员工技能提升率：[提升率]%年
- 知识资产价值：[价值]亿元
- 培训ROI：[ROI]倍
- 员工满意度：[满意度]%
```

**强制检查点**：必须获取以上6项教学数据，缺失任何一项不得进入下一阶段。

---

## Phase 2: 检查清单验证（强制执行）

### 2.1 执行教学效果检查清单

**任务**: 使用检查清单方法，验证教学效果。

**检查清单项目**（10项）：
1. [ ] 培训目标明确且可量化
2. [ ] 培训内容与业务需求匹配
3. [ ] 培训方式多样化（线上+线下）
4. [ ] 培训师资专业且经验丰富
5. [ ] 培训效果可评估
6. [ ] 培训成果可转化
7. [ ] 培训成本可控
8. [ ] 培训时间合理
9. [ ] 培训反馈机制完善
10. [ ] 培训持续改进机制有效

**通过标准**：
- **优秀**：通过9-10项（≥90%）
- **良好**：通过8项（≥80%）
- **一般**：通过7项（≥70%）
- **不足**：通过<7项（<70%）

**输出格式**：
```
【检查清单验证】
- 通过项目数：[数量]/10
- 通过率：[通过率]%
- 未通过项目：[项目列表]
- 评估结果：[优秀/良好/一般/不足]
```

**强制检查点**：必须完成10项检查清单验证并给出通过率。

---

## Phase 3: 反向思考验证（强制执行）

### 3.1 分析教学失败场景

**任务**: 使用反向思考，分析教学可能失败的场景。

**反向思考维度**：
1. **培训无效场景**：如果培训无法提升员工技能，如何应对？
2. **成本失控场景**：如果培训成本大幅上升，如何应对？
3. **员工流失场景**：如果培训后员工流失，如何应对？
4. **知识过时场景**：如果培训知识快速过时，如何应对？

**输出格式**：
```
【反向思考验证】
- 培训无效应对：[应对方案]
- 成本失控应对：[应对方案]
- 员工流失应对：[应对方案]
- 知识过时应对：[应对方案]
```

**强制检查点**：必须分析4个失败场景并给出应对方案。

---

## Phase 4: 多学科思维模型验证（强制执行）

### 4.1 应用多学科思维模型

**任务**: 使用多学科思维模型，验证教学策略。

**思维模型维度**：
1. **心理学模型**：员工学习动机与行为改变
2. **经济学模型**：培训投入产出比与机会成本
3. **管理学模型**：组织学习与知识管理
4. **教育学模型**：成人学习理论与教学方法

**输出格式**：
```
【多学科思维模型验证】
- 心理学模型应用：[应用描述]
- 经济学模型应用：[应用描述]
- 管理学模型应用：[应用描述]
- 教育学模型应用：[应用描述]
```

**强制检查点**：必须应用4个学科思维模型并给出验证结果。

---

## Phase 5: 教学质量综合评分（强制执行）

### 5.1 计算教学质量总分

**任务**: 综合所有分析维度，计算教学质量总分。

**计算公式**：
```
教学质量总分 = (检查清单通过率 × 0.4) + (反向思考评分 × 0.3) + (多学科模型评分 × 0.3)
```

**评分标准**（0-10分）：
- **优质教学**：教学质量总分≥8分
- **良好教学**：教学质量总分≥7分
- **一般教学**：教学质量总分≥6分
- **低质教学**：教学质量总分<6分

**输出格式**：
```
【教学质量综合评分】
- 检查清单通过率：[通过率]%
- 反向思考评分：[评分]/10
- 多学科模型评分：[评分]/10
- 教学质量总分：[总分]/10
- 评估结果：[优质/良好/一般/低质]
```

**强制检查点**：必须计算教学质量总分并给出评估结果。

## Phase 6: 最终决策与风险提示（强制执行）

### 6.1 综合评估与决策

**任务**: 综合所有分析结果，给出最终教学投资建议。

**决策标准**：
- **BUY（投资教学）**：教学质量总分≥8分、检查清单通过率≥80%、培训ROI≥3倍
- **HOLD（观察教学）**：教学质量总分≥7分、检查清单通过率≥70%、培训ROI≥2倍
- **SELL（优化教学）**：教学质量总分<7分、检查清单通过率<70%、培训ROI<2倍

**输出格式**：
```
【最终决策】
- 决策：[BUY/HOLD/SELL]
- 教学质量总分：[总分]/10
- 检查清单通过率：[通过率]%
- 培训ROI：[ROI]倍
- 核心理由：[理由描述]

【操作建议】
- 教学策略：[策略建议]
- 成本策略：[策略建议]
- 效果策略：[策略建议]
- 改进策略：[策略建议]
```

**强制检查点**：必须给出明确的BUY/HOLD/SELL决策和具体操作建议。

### 6.2 风险提示

**任务**: 提供教学投资的风险提示。

**输出格式**：
```
【风险提示】
- 主要风险因素：
  1. 培训效果风险：培训可能无法达到预期效果
  2. 成本风险：培训成本可能超出预算
  3. 员工流失风险：培训后员工可能离职
  4. 知识过时风险：培训知识可能快速过时

【免责声明】
本分析基于公开财务数据和教学评估信息，仅供参考，不构成任何投资建议。
教学投资有风险，决策需谨慎。过去的表现不代表未来的结果。
请结合自身业务实际情况做出独立判断。
```
