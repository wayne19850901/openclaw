## 工作流概述

本工作流基于 Cathie Wood 的颠覆性创新投资哲学，采用 **指数增长预测 + TAM分析 + 技术颠覆评估** 三大核心方法论。

**执行原则**：
1. **强制读完机制**：AI必须执行完整流程，不得跳过任何步骤
2. **数据源唯一性**：所有金融数据必须使用 `neodata-financial-search` skill
3. **量化指标验证**：每个决策必须有量化指标支撑（TAM≥1000亿、指数增长≥30%年增速）
4. **技术颠覆验证**：必须评估技术颠覆潜力（颠覆指数≥7分）

## Phase 1: 增长数据收集（强制执行）

### 1.1 获取目标公司增长数据

**任务**: 使用 `neodata-financial-search` skill 获取目标公司的增长相关数据。

**执行步骤**：
1. 调用 `neodata-financial-search` skill
2. 搜索目标公司的增长数据：
   - 收入增长率
   - 用户增长率
   - 市场份额增长率
   - 研发投入增长率
   - 市值增长率
   - TAM（总可触达市场）

**输出格式**：
```
【增长数据】
- 收入增长率：[增长率]%年
- 用户增长率：[增长率]%年
- 市场份额增长率：[增长率]%年
- 研发投入增长率：[增长率]%年
- 市值增长率：[增长率]%年
- TAM（总可触达市场）：[TAM]亿元
```

**强制检查点**：必须获取以上6项增长数据，缺失任何一项不得进入下一阶段。

## Phase 2: 指数增长预测（强制执行）

### 2.1 计算指数增长潜力

**任务**: 使用指数增长模型，预测未来5年的增长潜力。

**计算公式**：
```
指数增长预测 = 当前收入 × (1 + 年增长率)^5
```

**评估标准**：
- **指数增长**：年增长率≥30%（5年增长≥3.7倍）
- **高速增长**：年增长率≥20%（5年增长≥2.5倍）
- **中速增长**：年增长率≥10%（5年增长≥1.6倍）
- **低速增长**：年增长率<10%（5年增长<1.6倍）

**输出格式**：
```
【指数增长预测】
- 当前收入：[收入]亿元
- 年增长率：[增长率]%
- 5年后收入预测：[预测]亿元
- 增长倍数：[倍数]
- 增长类型：[指数/高速/中速/低速]
```

**强制检查点**：必须计算指数增长预测并给出评估结果。

## Phase 3: TAM分析（强制执行）

### 3.1 评估总可触达市场规模

**任务**: 评估目标公司的总可触达市场规模（TAM）。

**评估标准**：
- **巨大市场**：TAM≥1000亿元（市场空间巨大）
- **大型市场**：TAM≥500亿元（市场空间较大）
- **中型市场**：TAM≥100亿元（市场空间中等）
- **小型市场**：TAM<100亿元（市场空间有限）

**输出格式**：
```
【TAM分析】
- TAM规模：[规模]亿元
- 市场渗透率：[渗透率]%
- 市场增长潜力：[潜力描述]
- 市场类型：[巨大/大型/中型/小型]
```

**强制检查点**：必须评估TAM规模并给出评估结果。

## Phase 4: 技术颠覆评估（强制执行）

### 4.1 评估技术颠覆潜力

**任务**: 评估目标公司的技术颠覆潜力。

**评估维度**：
1. **技术突破性**：技术的突破程度（0-10分）
2. **市场颠覆性**：对现有市场的颠覆程度（0-10分）
3. **竞争壁垒**：技术壁垒的高度（0-10分）
4. **应用广度**：技术的应用范围（0-10分）

**评分标准**（0-10分）：
- **颠覆性技术**：颠覆指数≥8分
- **创新性技术**：颠覆指数≥7分
- **改进性技术**：颠覆指数≥6分
- **传统技术**：颠覆指数<6分

**计算公式**：
```
颠覆指数 = (技术突破性 + 市场颠覆性 + 竞争壁垒 + 应用广度) / 4
```

**输出格式**：
```
【技术颠覆评估】
- 技术突破性评分：[评分]/10
- 市场颠覆性评分：[评分]/10
- 竞争壁垒评分：[评分]/10
- 应用广度评分：[评分]/10
- 颠覆指数：[指数]/10
- 技术类型：[颠覆性/创新性/改进性/传统]
```

**强制检查点**：必须评估技术颠覆潜力并给出评分。

## Phase 5: 增长质量综合评分（强制执行）

### 5.1 计算增长质量总分

**任务**: 综合所有分析维度，计算增长质量总分。

**计算公式**：
```
增长质量总分 = (指数增长评分 × 0.3) + (TAM评分 × 0.3) + (技术颠覆指数 × 0.4)
```

**评分标准**（0-10分）：
- **优质增长**：增长质量总分≥8分
- **良好增长**：增长质量总分≥7分
- **一般增长**：增长质量总分≥6分
- **低质增长**：增长质量总分<6分

**输出格式**：
```
【增长质量综合评分】
- 指数增长评分：[评分]/10
- TAM评分：[评分]/10
- 技术颠覆指数：[指数]/10
- 增长质量总分：[总分]/10
- 评估结果：[优质/良好/一般/低质]
```

**强制检查点**：必须计算增长质量总分并给出评估结果。

## Phase 6: 最终决策与风险提示（强制执行）

### 6.1 综合评估与决策

**任务**: 综合所有分析结果，给出最终增长投资建议。

**决策标准**：
- **BUY（投资增长）**：增长质量总分≥8分、TAM≥1000亿、技术颠覆指数≥7分
- **HOLD（观察增长）**：增长质量总分≥7分、TAM≥500亿、技术颠覆指数≥6分
- **SELL（放弃增长）**：增长质量总分<7分、TAM<500亿、技术颠覆指数<6分

**输出格式**：
```
【最终决策】
- 决策：[BUY/HOLD/SELL]
- 增长质量总分：[总分]/10
- TAM规模：[规模]亿元
- 技术颠覆指数：[指数]/10
- 核心理由：[理由描述]

【操作建议】
- 增长策略：[策略建议]
- 技术策略：[策略建议]
- 市场策略：[策略建议]
- 竞争策略：[策略建议]
```

**强制检查点**：必须给出明确的BUY/HOLD/SELL决策和具体操作建议。

### 6.2 风险提示

**任务**: 提供增长投资的风险提示。

**输出格式**：
```

- 主要风险因素：

  3. 监管政策风险：监管政策可能限制技术应用
```
