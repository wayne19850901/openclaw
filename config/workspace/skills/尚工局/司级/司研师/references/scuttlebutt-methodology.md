# Scuttlebutt调研方法论

> **用途**: Philip Fisher产品研究核心方法论
> **适用场景**: 产品研究、市场调研、竞争分析
> **更新时间**: 2026-05-21

---

## 一、Scuttlebutt定义

**Scuttlebutt（闲聊调研法）**：通过非正式渠道（客户、供应商、竞争对手、员工等）收集信息，深入了解公司真实经营状况的调研方法。

**核心原则**：
- 信息来源多样化
- 避免单一信息源偏差
- 重视定性信息
- 关注细节和趋势

---

## 二、Scuttlebutt调研对象

### 2.1 客户访谈

**访谈目的**：
- 了解客户对产品的真实评价
- 发现产品的痛点和改进需求
- 评估客户忠诚度和推荐意愿

**访谈问题清单**：
1. 您为什么选择这个产品？
2. 您最满意的功能是什么？
3. 您最不满意的功能是什么？
4. 您会向朋友推荐这个产品吗？
5. 您认为这个产品与竞品相比如何？
6. 您未来会继续使用这个产品吗？
7. 您希望产品增加什么功能？
8. 您对产品的价格满意吗？
9. 您使用产品的频率如何？
10. 您对客户服务满意吗？

**访谈样本要求**：
- 至少访谈10个客户
- 覆盖不同类型客户（大客户、中小客户、新客户、老客户）
- 记录访谈时间和方式

---

### 2.2 供应商访谈

**访谈目的**：
- 了解公司的供应链稳定性
- 评估公司的付款信誉
- 判断公司的业务增长潜力

**访谈问题清单**：
1. 您与该公司合作多久了？
2. 该公司的订单量是否稳定？
3. 该公司的付款是否及时？
4. 该公司对产品质量要求如何？
5. 您认为该公司的业务前景如何？
6. 该公司是否是您的优质客户？
7. 您对该公司的合作满意度如何？
8. 该公司是否按时提货？
9. 该公司是否有拖欠货款的情况？
10. 您认为该公司的管理水平如何？

**访谈样本要求**：
- 至少访谈5个供应商
- 覆盖不同类型供应商（原材料、零部件、服务）
- 记录访谈时间和方式

---

### 2.3 竞争对手分析

**分析目的**：
- 了解市场竞争格局
- 识别竞争对手的优势和劣势
- 发现市场机会和威胁

**分析维度**：
1. **产品对比**：功能、性能、价格
2. **市场份额**：市场占有率、增长趋势
3. **竞争优势**：技术、品牌、渠道、成本
4. **竞争策略**：定价策略、营销策略、产品策略

**分析样本要求**：
- 至少分析5个竞争对手
- 覆盖直接竞争对手和间接竞争对手
- 收集公开信息和行业报告

## 三、Scuttlebutt调研流程

### 3.1 准备阶段

**任务清单**：
1. [ ] 确定调研目标公司
2. [ ] 收集公司基本信息
3. [ ] 制定访谈提纲
4. [ ] 确定访谈对象
5. [ ] 安排访谈时间

### 3.2 执行阶段

**任务清单**：
1. [ ] 执行客户访谈（至少10个）
2. [ ] 执行供应商访谈（至少5个）
3. [ ] 执行竞争对手分析（至少5个）
4. [ ] 收集行业信息和报告
5. [ ] 整理访谈记录

### 3.3 分析阶段

**任务清单**：
1. [ ] 整理访谈数据
2. [ ] 提取关键信息
3. [ ] 识别共同主题
4. [ ] 验证信息准确性
5. [ ] 形成调研报告

---

## 四、Scuttlebutt调研技巧

### 4.1 访谈技巧

1. **建立信任**：
   - 自我介绍和说明来意
   - 承诺保密信息
   - 尊重对方时间

2. **提问技巧**：
   - 开放式问题为主
   - 避免引导性问题
   - 追问细节和原因

3. **记录技巧**：
   - 全程录音（征得同意）
   - 及时记录关键信息
   - 标注信息来源

### 4.2 信息验证

**交叉验证方法**：
1. 对比不同来源的信息
2. 验证信息的一致性
3. 识别矛盾和异常信息
4. 追踪信息来源

## 六、案例库

### 6.1 成功案例

**苹果公司**
- Scuttlebutt调研发现：
  - 客户满意度极高（9/10）
  - 供应商合作稳定
  - 竞争对手难以模仿
- 投资建议：BUY
- 结果：股价10年增长10倍

### 6.2 失败案例

**某电动车公司**
- Scuttlebutt调研发现：
  - 客户满意度低（5/10）
  - 供应商投诉多
  - 竞争对手技术更优
- 投资建议：SELL
- 结果：股价下跌80%
