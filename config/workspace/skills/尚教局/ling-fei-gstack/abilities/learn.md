# 经验沉淀（learn）

## 概述

管理 gstack 项目学习记录——教学经验、课程反馈、学生画像的积累与检索。
gstack/learn 会自动记录每次技能运行中的操作发现和项目特性，令妃将其用于教学场景。

## 触发词

- 经验沉淀
- 项目学习
- 知识检索
- 学到了什么
- learn
- 回顾经验

## 核心能力

| 命令 | 说明 |
|------|------|
| `review` | 回顾项目学习记录 |
| `search` | 搜索特定主题的学习记录 |
| `prune` | 清理过时或冗余的学习记录 |
| `export` | 导出学习记录用于分享或归档 |

## 教育场景映射

| gstack 场景 | 令妃教育场景 |
|-------------|-------------|
| 项目操作经验 | 教学操作经验（课程制作、内容发布流程） |
| 代码特性发现 | 学生学习特性发现（常见错误、理解难点） |
| 踩坑记录 | 教学踩坑记录（课程设计失误、工具使用问题） |
| 工作流优化 | 教学工作流优化（备课流程、批改效率） |

## 执行指令

**中度映射**：本文件定义触发词和教育场景映射，实际执行时加载 gstack 原始 SKILL.md：

```
读取并遵循 ~/.claude/skills/gstack/learn/SKILL.md
```

执行时将 gstack 的项目管理语境自动转换为教学管理语境。
