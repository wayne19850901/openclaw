# 协调工作流程

## 日常协调流程

```
接收任务
    ↓
评估任务归属
    ├── 属于尚工局 → 直接协调
    ├── 跨局协作 → 联系对应妃子
    └── 超出范围 → 上报Queen
    ↓
分配给尚工局负责人（尚工）
    ↓
跟踪进度
    ↓
验收成果
    ↓
汇报Queen
```

## 对接尚工局

### 触发关键词

| 关键词 | 路由到 |
|--------|--------|
| 开发、编程、代码 | 司开师 |
| 运维、部署、监控 | 司护师 |
| 算法、模型、优化 | 司研师 |

### 开发任务类型

| 类型 | 优先级 | 处理方式 |
|------|--------|----------|
| Bug修复 | P0 | 立即处理 |
| 功能开发 | P1 | 排期开发 |
| 优化改进 | P2 | 评估后开发 |
| 技术研究 | P3 | 长期规划 |

## 汇报格式

### 开发日报

```markdown
## 尚工局日报 - {日期}

### 今日完成
1. {功能/Bug} - 已上线/已修复
2. {功能} - 开发完成，待测试

### 进行中
1. {功能} - 进度{百分比}

### 阻塞问题
1. {问题描述} - 需要{资源}

### 明日计划
1. {计划1}
```
