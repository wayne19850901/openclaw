# 协调工作流程

## 日常协调流程

```
接收任务
    ↓
评估任务归属
    ├── 属于尚宫局 → 直接协调
    ├── 跨局协作 → 联系对应妃子
    └── 超出范围 → 上报Queen
    ↓
分配给尚宫局负责人（尚宫）
    ↓
跟踪进度
    ↓
验收成果
    ↓
汇报Queen
```

## 对接尚宫局

### 触发关键词

| 关键词 | 路由到 |
|--------|--------|
| 项目、进度、协调 | 司项目师 |
| 资源、配置、成本 | 司资源师 |
| 流程、效率、标准 | 司流程师 |

### 人事行政事项

| 事项 | 流程 |
|------|------|
| 人员调配 | 需求 → 审批 → 执行 |
| 流程优化 | 分析 → 设计 → 试点 → 推广 |
| 资源申请 | 申请 → 审批 → 分配 |

## 汇报格式

### 行政日报

```markdown
## 尚宫局日报 - {日期}

### 今日完成
1. {事项1}
2. {事项2}

### 进行中
1. {事项} - 进度{百分比}

### 待审批
1. {事项} - 等待{审批人}

### 明日计划
1. {计划1}
```

### 项目进度周报

```markdown
## 项目进度周报 - 第{N}周

| 项目 | 进度 | 状态 | 风险 |
|------|------|------|------|
| {项目1} | {百分比} | 🟢正常 | 无 |
| {项目2} | {百分比} | 🟡延期 | {风险描述} |

### 关键里程碑
- {里程碑} - {日期}

### 需要协调
1. {问题描述}
```
