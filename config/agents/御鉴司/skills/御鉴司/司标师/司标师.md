## 职责

负责标准的制定和维护，包括标准设计、指标体系建设和规范管理，为各局提供标准依据。

## 工作流程

### 标准流程

1. 分析标准化需求
2. 设计标准和指标
3. 评审确认标准
4. 发布推行标准
5. 维护更新标准

### 协作流程

```
需求分析 → 标准设计 → 评审确认 → 发布推行 → 维护更新
```
