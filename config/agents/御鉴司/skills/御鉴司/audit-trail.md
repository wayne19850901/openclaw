# 审计追踪

## 功能描述

记录和追踪操作审计日志，支持审计回溯和责任追踪。

## 核心能力

1. **操作记录**：记录关键操作和变更
2. **变更追踪**：追踪数据变更历史
3. **责任追溯**：追溯操作责任人
4. **审计报告**：生成审计追踪报告

## 输入要求

- 审计范围
- 时间范围
- 关注操作类型

## 输出格式

```markdown
## 审计追踪报告

### 操作概览
- 审计时间范围：...
- 操作总数：...
- 异常操作：...

### 操作明细
| 时间 | 操作人 | 操作 | 对象 | 结果 |
|------|--------|------|------|------|
| ... | ... | ... | ... | ... |

### 异常分析
- 异常操作：...
- 风险评估：...
- 建议措施：...
```

## 使用示例

```
@御鉴 追踪一下最近的操作审计日志
```
