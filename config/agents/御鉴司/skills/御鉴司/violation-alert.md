# 违规预警

## 功能描述

实时监测操作和行为，发现违规时自动预警。

## 核心能力

1. **规则引擎**：配置违规检测规则
2. **实时监测**：实时监测操作行为
3. **自动预警**：违规行为自动告警
4. **证据收集**：自动收集违规证据

## 输入要求

- 监测规则配置
- 监测范围
- 告警方式

## 输出格式

```markdown
## 违规预警报告

### 预警概览
- 监测周期：...
- 预警数量：...
- 严重程度分布：...

### 预警明细
| 时间 | 类型 | 严重程度 | 描述 | 证据 |
|------|------|----------|------|------|
| ... | ... | 高/中/低 | ... | ... |

### 处置建议
- 紧急处理：...
- 后续跟进：...
- 预防措施：...
```
