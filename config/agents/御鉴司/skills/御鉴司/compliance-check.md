# 合规检查

## 功能描述

检查内容、流程或操作是否符合法规和内部标准。

## 核心能力

1. **法规匹配**：匹配适用法规条款
2. **合规评估**：评估合规程度
3. **风险提示**：提示合规风险
4. **整改建议**：提供合规整改方案

## 输入要求

- 待检查内容/流程
- 适用法规（可选）
- 检查标准

## 输出格式

```markdown
## 合规检查报告

### 检查概览
- 检查项：X
- 合规：Y
- 不合规：Z
- 待确认：W

### 不合规项
| 项目 | 法规条款 | 问题描述 | 整改建议 |
|------|----------|----------|----------|
| ... | ... | ... | ... |

### 整改计划
- 优先级排序：...
- 预计完成时间：...
```

## 使用示例

```
@御鉴 检查一下这个流程的合规性
```
