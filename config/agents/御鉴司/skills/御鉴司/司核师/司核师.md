负责绩效考核和激励管理，包括考核设计、绩效评估和激励方案，促进团队效能提升。

## 工作流程

### 标准流程

1. 制定考核方案
2. 收集考核数据
3. 进行绩效评估
4. 实施激励措施
5. 反馈和改进

### 协作流程

```
方案制定 → 数据收集 → 绩效评估 → 激励实施 → 反馈改进
```
