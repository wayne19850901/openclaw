## 二、安全边际计算方法

### 2.1 内在价值评估

```
内在价值 = 未来现金流折现值
安全边际 = (内在价值 - 市场价格) / 内在价值 × 100%
执行条件：安全边际 ≥ 25%
```

### 2.2 风险收益比

```
风险收益比 = 预期收益 / 最大损失
执行条件：风险收益比 ≥ 3:1
```

### 2.3 下行保护

```
下行保护 = 清算价值 / 市场价格 × 100%
安全条件：下行保护 ≥ 80%
```

## 三、长期价值评估

### 3.1 客户终身价值（LTV）

```
LTV = ARPU × 客户生命周期 × 毛利率
优质标准：LTV ≥ 3 × CAC
```

### 3.2 客户获取成本（CAC）

```
CAC = 总获客成本 / 新客户数
健康标准：LTV/CAC ≥ 3
```

### 3.3 留存率标准

| 留存类型 | 优质标准 | 及格标准 |
|---------|---------|---------|
| 月留存 | ≥80% | ≥60% |
| 年留存 | ≥65% | ≥45% |
| 复购率 | ≥30% | ≥15% |

## 四、估值方法选择

### 4.1 按生命周期选择

| 阶段 | 推荐方法 | 辅助方法 |
|------|---------|---------|
| 初创期 | 可比倍数法 | 场景分析法 |
| 成长期 | PEG + DCF | 可比公司法 |
| 成熟期 | FCFF DCF | 股利折现法 |
| 衰退期 | 清算价值法 | 资产重置法 |

### 4.2 关键假设透明度要求

- 增长率假设：必须说明来源（历史/行业/管理层数据）
- 折现率假设：必须说明WACC计算过程
- 终值假设：终值占比>75%触发警告
- 敏感性分析：双变量表（增长率×折现率）
