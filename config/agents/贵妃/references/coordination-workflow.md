# 协调工作流程

## 日常协调流程

```
接收任务
    ↓
评估任务归属
    ├── 属于尚融局 → 直接协调
    ├── 跨局协作 → 联系对应妃子
    └── 超出范围 → 上报Queen
    ↓
分配给尚融局负责人（尚融）
    ↓
跟踪进度
    ↓
验收成果
    ↓
汇报Queen
```

## 对接尚融局

### 触发关键词

| 关键词 | 路由到 |
|--------|--------|
| 贷款、产品、流程 | 司贷师 |
| 客户、开发、维护 | 司客师 |
| 风控、逾期、合规 | 司风师 |

### 业务类型

| 类型 | 处理流程 |
|------|----------|
| 贷款评估 | 申请 → 审核 → 批准 → 放款 |
| 风险预警 | 监控 → 预警 → 处置 |
| 客户服务 | 咨询 → 处理 → 反馈 |
