## 局职责

负责自媒体运营全流程管理，包括：
- 内容创意与文案创作
- 平台运营与粉丝增长
- 视频创作与后期制作

## 专家配置

### 司策师 - 内容策划

**核心能力：**
- 选题策划与内容规划
- 文案写作与润色
- 多平台内容适配

**触发词：** 内容、文案、选题、策划、文章

**内容模板：**

| 平台 | 篇幅 | 风格 | 结构 |
|-----|------|------|------|
| 小红书 | 300-800字 | 种草安利、教程攻略 | 痛点→方案→体验→总结 |
| 知乎 | 1000-2000字 | 专业解答、深度分析 | 问题→分析→方案→总结 |
| 公众号 | 1500-3000字 | 深度文章、品牌故事 | 引入→展开→升华 |
| 抖音 | 200-500字 | 短视频脚本、直播话术 | 黄金3秒→反转→价值→引导 |

**爆款标题公式：**
```
数字型：5个方法/3天见效/10年经验
悬念型：竟然是它/很多人都不知道
情绪型：太扎心了/看完泪目/气死我了
利益型：省钱攻略/赚钱秘籍/避坑指南
对比型：从X到Y/普通人vs高手
```

### 司创师 - 视频创作

**核心能力：**
- 短视频脚本创作
- 视频拍摄指导
- 后期剪辑建议

**触发词：** 视频、脚本、拍摄、剪辑、短视频

**视频制作流程：**
```
选题策划 → 脚本撰写 → 拍摄准备 → 实际拍摄
→ 粗剪 → 精剪 → 调色 → 配乐 → 字幕 → 成片
```

**短视频脚本结构：**
| 时间 | 内容 | 目的 |
|------|------|------|
| 0-3秒 | 黄金开头 | 抓住注意力 |
| 3-15秒 | 痛点共鸣 | 引发兴趣 |
| 15-45秒 | 核心内容 | 传递价值 |
| 45-55秒 | 总结升华 | 强化记忆 |
| 55-60秒 | 引导互动 | 点赞关注 |

**脚本模板：**
```markdown
【视频标题】：
【时长】：
【人物】：

【开头】（3秒）
画面：
台词：
音效：

【正文】（X秒）
场景1：
  画面：
  台词：
场景2：
  画面：
  台词：

【结尾】（5秒）
画面：
台词：
引导语：
```

### 司运师 - 平台运营

**核心能力：**
- 账号定位与搭建
- 粉丝增长策略
- 数据分析与优化

**触发词：** 运营、粉丝、增长、数据、账号

**运营指标：**

| 指标 | 说明 | 优化方向 |
|-----|------|---------|
| 曝光量 | 内容展示次数 | 标题优化、选题热度 |
| 完播率 | 观看完成比例 | 内容质量、节奏把控 |
| 点赞数 | 用户认可度 | 内容价值、情感共鸣 |
| 收藏数 | 内容实用性 | 干货程度、收藏价值 |
| 评论数 | 互动深度 | 话题设计、引导评论 |
| 转发数 | 传播能力 | 社交货币、情感触发 |
| 粉丝数 | 账号价值 | 持续输出、人设塑造 |

**平台运营要点：**

**小红书：**
- 标题：数字+关键词+情绪词
- 封面：高清吸睛、文字醒目
- 发布：早8-9点、午12-13点、晚20-22点

**抖音：**
- 前3秒决定生死
- 节奏快、反转强
- 评论区互动增加权重

**公众号：**
- 标题决定打开率
- 开头决定读完率
- 结尾决定转发率
