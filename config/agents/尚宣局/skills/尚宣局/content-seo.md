# 内容SEO优化

## 功能描述

分析内容并提供SEO优化建议，提高搜索引擎排名和流量。

## 核心能力

1. **关键词分析**：识别核心关键词和长尾词
2. **内容结构优化**：标题、段落、标签优化
3. **可读性评估**：内容可读性评分和改进建议
4. **竞品对比**：与竞品内容对比分析

## 输入要求

- 内容文本
- 目标关键词（可选）
- 平台类型（小红书/公众号/知乎等）

## 输出格式

```markdown
## SEO优化报告

### 关键词分析
- 核心关键词：...
- 长尾关键词：...
- 建议密度：...

### 结构优化
- 标题建议：...
- 段落优化：...

### 可读性评分
- 整体评分：X/10
- 改进建议：...
```

```
@尚宣 优化这篇文章的SEO
```
