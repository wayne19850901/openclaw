## 工作流程

### 选题流程

1. **热点收集** - 监控热点、行业动态
2. **选题筛选** - 结合账号定位筛选
3. **价值评估** - 评估流量价值和执行难度
4. **排期规划** - 制定发布时间表
5. **效果追踪** - 跟踪选题效果数据

### 选题评估维度

| 维度 | 说明 | 权重 |
|------|------|------|
| 热度 | 话题搜索量、讨论度 | 30% |
| 相关性 | 与账号定位的匹配度 | 25% |
| 时效性 | 话题的最佳发布窗口 | 20% |
| 竞争度 | 同类内容的数量和质量 | 15% |
| 执行难度 | 内容制作的复杂程度 | 10% |

### 选题类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 热点型 | 时效强、流量大 | 追热点、蹭流量 |
| 干货型 | 实用性强、长尾流量 | 知识分享、教程 |
| 情感型 | 共鸣强、易传播 | 情感故事、观点 |
| 争议型 | 话题性强、互动高 | 观点表达、讨论 |
| 工具型 | 收藏价值高 | 资源分享、工具推荐 |

## 常见场景

### 场景1：热点选题判断

**热点事件：** XX事件上了热搜

**判断要点：**
1. 与账号定位是否相关
2. 正面/负面/中性
3. 时效窗口多长
4. 同类内容竞争情况
5. 自身差异化角度

### 场景2：月度选题规划

**需求：** 制定下个月的内容选题

**输出：**
1. 月度内容主题
2. 每周选题方向
3. 具体选题清单
4. 发布时间安排
