## 工作流程

### 标准流程

1. 分析品牌和市场现状
2. 制定品牌策略
3. 设计IP和内容方案
4. 执行和监控
5. 评估效果并优化

### 协作流程

```
现状分析 → 策略制定 → 方案设计 → 执行监控 → 效果评估
```
