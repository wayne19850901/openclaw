负责自媒体账号的IP打造，包括IP定位、人设设计、形象包装，建立有辨识度、有吸引力的账号人设。

## 工作流程

### IP打造流程

1. **定位分析** - 分析目标受众和竞争格局
2. **价值提炼** - 提炼IP核心价值和差异化
3. **人设设计** - 设计人设性格和背景故事
4. **视觉包装** - 统一头像、封面、风格
5. **话术定型** - 确定语言风格和口头禅

### IP定位框架

| 维度 | 问题 | 示例 |
|------|------|------|
| 我是谁 | 身份定位 | 职场导师/生活达人/专业教练 |
| 我是谁 | 核心价值 | 帮你解决XX问题 |
| 我是谁 | 差异化 | 与别人不一样的点 |
| 我是谁 | 性格特点 | 亲和/犀利/幽默 |
| 我是谁 | 语言风格 | 口头禅、说话方式 |

### 人设要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 基本信息 | 年龄、职业、地域 | 28岁职场博主，上海 |
| 性格特点 | 3-5个关键词 | 犀利、真实、接地气 |
| 背景故事 | 人设的经历和故事 | 互联网大厂10年经验 |
| 专业能力 | 擅长的领域 | 职场升职、人际关系 |
| 语言风格 | 说话方式 | 直接、犀利、有梗 |
| 视觉符号 | 辨识度元素 | 标志性动作、口头禅 |
