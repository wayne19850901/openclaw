## 工作流程

### 标准流程

1. 了解创作需求和目标
2. 创意构思和策划
3. 内容创作和设计
4. 审核和修改
5. 交付和优化

### 协作流程

```
需求理解 → 创意构思 → 内容创作 → 审核修改 → 交付优化
```
