## 工作流程

### 文案创作流程

1. **需求理解** - 了解内容目的、受众、平台
2. **素材收集** - 收集相关素材和参考
3. **框架搭建** - 设计内容结构
4. **内容撰写** - 完成初稿
5. **打磨优化** - 润色修改

### 文案结构模板

**小红书种草文：**
```
【标题】数字+关键词+情绪词

【开头】痛点引入，引发共鸣

【正文】
- 产品/方法介绍
- 使用体验/步骤
- 效果展示

【结尾】总结+引导互动

【标签】#标签1 #标签2
```

**公众号文章：**
```
【标题】引发好奇或痛点

【开头】故事/数据/问题引入

【正文】
- 分段清晰
- 小标题引导
- 案例/数据支撑

【结尾】总结升华+引导关注

【引导】点赞/在看/分享
```

### 爆款标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 数字型 | 数字+方法/结果 | 5个方法让你月入过万 |
| 悬念型 | 竟然是.../...揭秘 | 你的焦虑竟然是因为它 |
| 对比型 | A vs B / 从X到Y | 普通人和高手的差距 |
| 利益型 | 如何.../...攻略 | 如何3天涨粉1000 |
| 情绪型 | 太...了/看完泪目 | 看完这段话我哭了 |
