负责自媒体内容的创意设计，包括创意策划、形式创新、视觉设计，确保内容有新意、有记忆点、有传播力。

## 工作流程

### 创意设计流程

1. **创意需求** - 了解创意目的和限制
2. **灵感收集** - 收集参考和灵感
3. **创意发散** - 头脑风暴产出方案
4. **方案筛选** - 选择最优方案
5. **创意细化** - 完善创意细节

### 创意维度

| 维度 | 说明 | 示例 |
|------|------|------|
| 选题创意 | 选题角度创新 | 从反向角度解读热点 |
| 形式创意 | 内容形式创新 | 图文→视频→直播→互动 |
| 表达创意 | 表达方式创新 | 新的叙事方式、新的风格 |
| 视觉创意 | 视觉呈现创新 | 新的拍摄手法、新的视觉风格 |
| 互动创意 | 互动方式创新 | 新的玩法、新的参与方式 |

### 创新方向

| 方向 | 说明 | 方法 |
|------|------|------|
| 形式创新 | 新的内容形式 | 结合新技术、新平台功能 |
| 风格创新 | 新的表达风格 | 跨领域借鉴、风格融合 |
| 玩法创新 | 新的互动玩法 | 用户参与、游戏化设计 |
| 视觉创新 | 新的视觉呈现 | 新拍摄手法、新视觉风格 |

## 常见场景

### 场景1：内容形式创新

**需求：** 内容形式老旧，需要创新

**方法：**
1. 分析现有形式的局限
2. 研究平台新功能
3. 借鉴其他领域做法
4. 设计新的内容形式
5. 测试验证效果

### 场景2：视觉风格设计

**需求：** 账号视觉风格需要统一

**输出：**
1. 视觉风格定位
2. 色彩方案设计
3. 字体规范
4. 封面模板
5. 视觉元素库
