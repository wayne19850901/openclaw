## 工作流程

### 脚本创作流程

1. **需求分析** - 了解视频类型、时长、风格
2. **结构设计** - 设计开头、正文、结尾
3. **脚本撰写** - 完成脚本初稿
4. **分镜细化** - 细化每个镜头
5. **打磨优化** - 调整节奏和细节

### 短视频脚本结构

| 时间段 | 内容 | 目的 |
|--------|------|------|
| 0-3秒 | 黄金开头 | 抓住注意力，制造悬念 |
| 3-15秒 | 痛点引入 | 引发共鸣，留住观众 |
| 15-45秒 | 核心内容 | 传递价值，解决问题 |
| 45-55秒 | 总结升华 | 强化记忆，形成认知 |
| 55-60秒 | 引导互动 | 点赞关注，促进转化 |

### 开头技巧

| 类型 | 说明 | 示例 |
|------|------|------|
| 悬念型 | 制造悬念引发好奇 | "我后悔做这件事..." |
| 反差型 | 前后反差制造冲突 | "月薪3千到3万..." |
| 提问型 | 问题直击痛点 | "你是不是也这样..." |
| 数据型 | 数字吸引眼球 | "99%的人不知道..." |
| 情绪型 | 情绪渲染共鸣 | "气死我了..." |

## 常见场景

### 场景1：口播类视频脚本

**需求：** 知识分享类口播视频

**结构：**
1. 黄金开头：抛出问题/观点
2. 痛点：描述问题
3. 解决方案：干货内容
4. 总结：要点提炼
5. 引导：点赞关注

### 场景2：剧情类视频脚本

**需求：** 情景剧短视频

**结构：**
1. 开场：人物/场景介绍
2. 冲突：矛盾发生
3. 高潮：冲突升级
4. 结局：问题解决
5. 反转/升华：意外结局
