# 品牌语调检查

## 功能描述

检查内容是否符合品牌调性，确保品牌形象一致性。

## 核心能力

1. **语调分析**：分析内容语调特征
2. **品牌匹配**：与品牌调性对比
3. **一致性检查**：跨内容一致性评估
4. **修改建议**：提供具体修改建议

## 输入要求

- 待检查内容
- 品牌调性定义（可选）
- 参考内容（可选）

## 输出格式

```markdown
## 品牌语调检查报告

### 语调分析
- 当前语调：...
- 目标语调：...

### 一致性评分
- 整体一致性：X/10
- 问题点：...

### 修改建议
- 原文：...
- 建议：...
```

## 使用示例

```
@尚宣 检查这篇内容的品牌语调
```
