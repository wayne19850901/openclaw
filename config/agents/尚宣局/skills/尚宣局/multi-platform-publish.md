# 多平台发布适配

## 功能描述

将同一内容适配多个发布平台，自动调整格式、长度和风格。

## 核心能力

1. **平台规则适配**：根据平台规则调整内容
2. **格式转换**：自动调整标题、段落、标签
3. **长度优化**：根据平台限制调整内容长度
4. **风格适配**：调整语调风格匹配平台用户

## 输入要求

- 原始内容
- 目标平台列表
- 保留要点（可选）

## 输出格式

```markdown
## 多平台适配报告

### 小红书版本
- 标题：...
- 内容：...（已适配）

### 公众号版本
- 标题：...
- 内容：...（已适配）

### 知乎版本
- 标题：...
- 内容：...（已适配）
```

## 使用示例

```
@尚宣 把这篇文章适配到小红书和公众号
```
