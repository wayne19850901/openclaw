## 职责

负责新媒体账号的运营管理，包括内容发布、流量获取和粉丝互动，提升账号影响力和商业价值。

## 工作流程

### 标准流程

1. 制定运营计划
2. 执行内容发布
3. 监控数据反馈
4. 优化运营策略
5. 总结复盘改进

### 协作流程

```
计划制定 → 内容发布 → 数据监控 → 策略优化 → 复盘改进
```
