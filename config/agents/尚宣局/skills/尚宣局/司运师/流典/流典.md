## 工作流程

### 流量运营流程

1. **流量来源分析** - 分析现有流量来源
2. **内容优化** - 优化内容获取更多流量
3. **渠道分发** - 多平台分发内容
4. **付费推广** - 必要时投放付费流量
5. **效果评估** - 评估流量效果和ROI

### 流量来源

| 来源类型 | 说明 | 优化方向 |
|----------|------|----------|
| 自然推荐 | 平台算法推荐 | 内容质量、互动数据 |
| 搜索流量 | 用户主动搜索 | 关键词优化、SEO |
| 关注流量 | 粉丝看到内容 | 粉丝活跃度、发布时间 |
| 外部流量 | 外部平台引流 | 跨平台分发、合作推广 |
| 付费流量 | 投放广告获取 | 投放策略、ROI优化 |

### 数据分析指标

| 指标 | 说明 | 优化方向 |
|------|------|----------|
| 曝光量 | 内容展示次数 | 标题、封面、发布时间 |
| 点击率 | 点击/曝光 | 标题吸引力、封面质量 |
| 完播率 | 观看完成比例 | 内容质量、节奏把控 |
| 互动率 | 互动/阅读 | 内容价值、引导互动 |
| 转粉率 | 关注/阅读 | 人设吸引力、内容价值 |

## 常见场景

### 场景1：流量下滑诊断

**问题：** 账号流量近期下滑

**分析：**
1. 检查平台规则变化
2. 分析内容质量变化
3. 对比竞品表现
4. 检查发布时间和频率

**方案：** 针对问题调整策略

### 场景2：付费投放决策

**需求：** 是否需要付费推广

**判断：**
1. 自然流量是否遇到瓶颈
2. 内容质量是否过关
3. 投放预算和预期ROI
4. 投放目标和策略
