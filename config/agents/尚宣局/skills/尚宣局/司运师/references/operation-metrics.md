# 运营指标速查

## 核心指标

### 用户指标
| 指标 | 定义 | 计算方式 |
|------|------|----------|
| DAU | 日活跃用户 | 当日登录用户数 |
| MAU | 月活跃用户 | 当月登录用户数 |
| 新增用户 | 新注册用户 | 新注册数 |
| 用户留存 | 回访比例 | n日后回访/新增 |

### 内容指标
| 指标 | 定义 | 行业基准 |
|------|------|----------|
| 阅读量 | 内容曝光 | - |
| 点赞率 | 互动质量 | 3-5% |
| 评论率 | 深度互动 | 0.5-1% |
| 分享率 | 传播力 | 0.1-0.5% |
| 收藏率 | 价值度 | 1-3% |

### 商业指标
| 指标 | 定义 | 计算方式 |
|------|------|----------|
| GMV | 成交总额 | 订单金额总和 |
| 转化率 | 购买转化 | 购买数/访问数 |
| 客单价 | 平均订单金额 | GMV/订单数 |
| ROI | 投资回报率 | 收益/成本 |

---

## 数据分析方法

### 对比分析
- 同比：与去年同期对比
- 环比：与上一周期对比
- 横向：与竞品对比

### 漏斗分析
```
曝光 → 点击 → 浏览 → 互动 → 转化
 ↓      ↓      ↓      ↓      ↓
 识别各环节流失点
```
