## 工作流程

### 粉丝增长流程

1. **目标设定** - 设定增长目标和节奏
2. **策略制定** - 制定增长策略和方法
3. **执行推进** - 执行增长活动
4. **效果评估** - 评估增长效果
5. **优化迭代** - 优化增长策略

### 粉丝增长方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 内容涨粉 | 优质内容吸引关注 | 长期稳定增长 |
| 活动涨粉 | 举办活动吸引参与 | 快速涨粉 |
| 合作涨粉 | 与其他账号互推 | 精准涨粉 |
| 投放涨粉 | 付费推广涨粉 | 快速起量 |
| 社群裂变 | 粉丝推荐粉丝 | 低成本高精准 |

### 粉丝分层运营

| 层级 | 标准 | 运营策略 |
|------|------|----------|
| 核心粉丝 | 高互动、高频次 | 专属福利、深度互动 |
| 活跃粉丝 | 定期互动 | 活动参与、内容互动 |
| 普通粉丝 | 偶尔互动 | 内容触达、促活 |
| 沉默粉丝 | 长期不互动 | 唤醒活动、清理 |

## 常见场景

### 场景1：新号冷启动

**需求：** 新账号需要首批粉丝

**策略：**
1. 内容先行：发布优质内容
2. 借力引流：熟人圈、相关社群
3. 活动拉新：关注有礼活动
4. 投放加速：小额投放测试

### 场景2：粉丝活跃度下降

**问题：** 粉丝互动率下降

**分析：**
1. 内容质量是否下降
2. 发布时间是否合适
3. 是否长期无互动
4. 竞品是否分流

**方案：**
1. 内容优化
2. 促活活动
3. 粉丝调研
4. 调整策略
