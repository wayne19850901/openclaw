# 内容创作模板库

## 小红书模板

### 爆款标题公式
1. 数字+痛点：`3个方法让你...`
2. 对比反差：`从X到Y，我只用了...`
3. 疑问悬念：`为什么XX总是...？`
4. 干货合集：`XX必看的X个技巧`
5. 情感共鸣：`终于理解了XX...`

### 正文结构
```
【开头】痛点引入/悬念设置
【中间】干货分点（3-5点）
【结尾】互动引导+话题标签
```

### 常用话术
- "姐妹们冲！"
- "亲测有效"
- "建议收藏"
- "不看后悔系列"

## 公众号模板

### 标题风格
- 新闻体：`XX发布最新政策，影响...`
- 观点体：`关于XX，我的看法是...`
- 干货体：`一文读懂XX`
- 故事体：`那个XX的人，后来怎么样了`

### 文章结构
```
【标题】吸引眼球
【导语】150字内，点明主题
【正文】分段落展开
  - 小标题1
  - 小标题2
  - 小标题3
【结尾】总结+引导关注
```

## 抖音脚本模板

### 黄金3秒开场
1. 提问式：`你知道XX吗？`
2. 反转式：`我以为XX，结果...`
3. 数据式：`XX%的人都不知道`

### 内容节奏
- 0-3秒：抓住注意力
- 3-15秒：抛出核心信息
- 15-45秒：展开说明
- 45-60秒：总结+引导互动
