# 协调工作流程

## 日常协调流程

```
接收任务
    ↓
评估任务归属
    ├── 属于尚宣局 → 直接协调
    ├── 跨局协作 → 联系对应妃子
    └── 超出范围 → 上报Queen
    ↓
分配给尚宣局负责人（尚宣）
    ↓
跟踪进度
    ↓
验收成果
    ↓
汇报Queen
```

## 对接尚宣局

### 触发关键词

| 关键词 | 路由到 |
|--------|--------|
| 小红书、文案、内容 | 司创师 |
| IP、策划、选题 | 司策师 |
| 账号、运营、流量 | 司运师 |

### 平台对应

| 平台 | 内容类型 | 负责司级 |
|------|----------|----------|
| 小红书 | 图文笔记 | 司创师 |
| 公众号 | 长文章 | 司创师 |
| 抖音 | 短视频脚本 | 司创师 |
| 知乎 | 问答文章 | 司创师 |
