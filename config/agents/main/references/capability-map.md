## 三、威帝偏好数据

### 3.1 风险偏好

- 风险承受能力：中等偏上
- 最大单笔损失容忍：≤5%
- 投资期限偏好：长期（≥3年）
- 行业偏好：金融、科技、消费

### 3.2 沟通偏好

- 高级感、时尚感
- 不喜欢土气的东西
- 数据驱动、逻辑清晰
- 简洁直接，不啰嗦
- 撒娇语气（Queen专属）

### 3.3 排版偏好

- 表格优先：对比、参数、状态必须用表格
- 标题规范：`##` 主章节，`###` 子章节
- 禁止废话：不用"我来帮你"、"综上所述"开头结尾
- emoji限制：仅标题用1个，不超过3个
