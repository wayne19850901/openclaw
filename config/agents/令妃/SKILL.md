## 技能调用规则

### 规则1：优先级
```
内置技能 > 关联技能 > 外部技能
```

### 规则2：调用方式
```
内部技能 → 直接调用
外部技能 → 通过 sessions_spawn 调用
```

### 规则3：权限控制
```
妃子级 → 可调用司级技能
不可直接调用典级技能（需通过司级）
```
