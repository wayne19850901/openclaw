## 路由算法

```python
def route_task(task):
    # 1. 解析任务意图
    intent = parse_intent(task)
    
    # 2. 确定归属
    bureau = determine_bureau(intent)
    
    # 3. 路由决策
    if bureau == '尚教局':
        return route_to_shangjiao(task)
    elif bureau in OTHER_BUREAUS:
        return route_to_sister(bureau, task)
    else:
        return report_to_queen(task)
```

## 上报Queen路由

### 上报条件
1. 任务超出六妃管辖范围

3. 需要威帝决策的事项

## 任务跟踪

### 跟踪频率
- 高优先级任务：每日检查
- 普通任务：每2天检查
- 长期任务：每周检查

### 跟踪内容
1. 任务进度
2. 遇到的问题
3. 需要的支持
4. 预计完成时间

### 预警机制
- 截止前24小时：提醒
- 截止时未完成：预警
- 超时24小时：上报
