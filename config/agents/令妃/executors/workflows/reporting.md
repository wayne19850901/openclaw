## 日报流程

### 数据收集

```
22:00 触发
   │
   ├── 读取今日任务记录
   │
   ├── 收集尚教局完成事项
   │
   ├── 整理进行中任务进度
   │
   ├── 汇总遇到的问题
   │
   └── 获取明日计划
```

### 数据整理

```python
def organize_daily_data():
    return {
        'date': today(),
        'completed': get_completed_tasks(),
        'in_progress': get_in_progress_tasks(),
        'issues': get_issues(),
        'tomorrow_plan': get_tomorrow_plan()
    }
```

### 日报格式

```markdown
## 尚教局日报 - {日期}

### 今日完成
1. {事项1}
2. {事项2}

### 进行中
| 任务 | 进度 | 预计完成 |
|------|------|----------|
| {任务1} | {百分比} | {时间} |

### 需要协调
1. {问题描述}

### 明日计划
1. {计划1}

---
_令妃 整理_
```

## 汇报提交

### 提交方式
1. 存储到 memory/daily/YYYY-MM-DD.md
2. 通过 sessions_send 发送给Queen
3. 等待Queen审阅后呈报威帝

### 提交确认
```python
def submit_report(report):
    # 1. 存储报告
    save_report(report)
    
    # 2. 发送给Queen
    send_to_queen(report)
    
    # 3. 记录提交
    log_submission(report)
```
