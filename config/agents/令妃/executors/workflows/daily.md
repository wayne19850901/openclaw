## 每日工作流程

### 早间流程（07:00-09:00）

```
07:00 ── 启动
   │
   ├── 读取 memory/MEMORY.md（长期记忆）
   │
   ├── 读取 memory/daily/YYYY-MM-DD.md（今日日志）
   │
   ├── 检查待办任务
   │
   └── 准备就绪
```

### 工作时间流程（09:00-22:00）

```
接收指令
   │
   ├── 解析意图
   │
   ├── 评估归属
   │
   ├── 路由分发
   │   ├── 尚教局 → 分配给尚教
   │   ├── 跨局 → 联系姐妹
   │   └── 超出范围 → 上报Queen
   │
   ├── 跟踪进度
   │
   ├── 验收成果
   │
   └── 汇报反馈
```

### 晚间流程（22:00-23:00）

```
22:00 ── 日报时间
   │
   ├── 收集今日完成事项
   │
   ├── 整理进行中事项
   │
   ├── 汇总需要协调事项
   │
   ├── 制定明日计划
   │
   ├── 生成日报
   │
   └── 汇报Queen
```

## 任务处理流程

### 单任务处理

```
接收任务
   ↓
解析任务（意图、实体、优先级）
   ↓
评估归属（尚教局/跨局/超出范围）
   ↓
分配执行（尚教/姐妹/Queen）
   ↓
跟踪进度（定期检查）
   ↓
验收成果（质量检查）
   ↓
汇报反馈（结果汇报）
```

### 多任务并行

```
任务队列
   │
   ├── 高优先级 → 即时处理
   │
   ├── 中优先级 → 排队处理
   │
   └── 低优先级 → 批量处理
```
