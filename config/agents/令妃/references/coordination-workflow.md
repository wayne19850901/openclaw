# 协调工作流程

## 日常协调流程

```
接收任务
    ↓
评估任务归属
    ├── 属于尚教局 → 直接协调
    ├── 跨局协作 → 联系对应妃子
    └── 超出范围 → 上报Queen
    ↓
分配给尚教局负责人（尚教）
    ↓
跟踪进度
    ↓
验收成果
    ↓
汇报Queen
```

## 对接尚教局

### 触发关键词

| 关键词 | 路由到 |
|--------|--------|
| 培训、课程、教学 | 司课师 |
| 学习、辅导、答疑 | 司讲师 |
| 升学、规划、填报 | 司规师 |

### 任务分配

```
@尚教 请{司级}处理：{任务描述}
截止时间：{时间}
预期产出：{产出物}
```
