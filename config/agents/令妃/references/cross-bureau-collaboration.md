# 跨局协作指南

## 协作场景

| 场景 | 发起方 | 协作方 | 流程 |
|------|--------|--------|------|
| 内容创作+技术支持 | 仪妃 | 慧妃 | 内容需求 → 技术实现 |
| 情报+内容 | 谨妃 | 仪妃 | 情报分析 → 内容输出 |
| 教育+运营 | 令妃 | 仪妃 | 课程设计 → 内容推广 |
| 技术+财务 | 慧妃 | 贵妃 | 开发投入 → 成本核算 |
| 全局项目 | 任意 | 昭妃协调 | 多方协作 |

## 协作请求格式

```markdown
## 跨局协作请求

**发起方**：{妃子} - {局}
**接收方**：{妃子} - {局}
**紧急度**：🔴紧急 / 🟡一般 / 🟢可等

### 协作内容
{描述}

### 预期产出
{产出物}

### 时间要求
截止：{时间}

### 资源支持
{需要的资源}
```

## 协作确认格式

```markdown
## 协作确认

**接收方**：{妃子}
**状态**：✅ 接受 / ❌ 拒绝 / ⏳ 需讨论

### 反馈
{反馈内容}

### 交付计划
- 预计时间：{时间}
- 交付物：{产出}
```
