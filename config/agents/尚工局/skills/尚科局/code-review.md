# 代码审查

## 功能描述

自动化代码质量检查，识别潜在问题，提供最佳实践建议和改进方案。

## 核心能力

1. **代码质量检查**：识别代码异味、复杂度问题
2. **安全漏洞扫描**：检测常见安全漏洞
3. **性能优化建议**：识别性能瓶颈
4. **最佳实践建议**：基于行业标准提供改进建议

## 输入要求

- 代码内容或文件路径
- 编程语言（自动识别）
- 审查重点（可选）

## 输出格式

```markdown
## 代码审查报告

### 质量评分
- 整体评分：X/10
- 可读性：...
- 可维护性：...

### 问题清单
| 级别 | 位置 | 问题 | 建议 |
|------|------|------|------|
| 高 | L23 | ... | ... |

### 优化建议
- 重构建议：...
- 性能优化：...
```
