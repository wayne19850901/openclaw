## 工作流程

### 优化流程

1. **问题诊断** - 分析性能瓶颈或效果问题
2. **方案设计** - 制定优化方案
3. **实施优化** - 执行优化措施
4. **效果验证** - 验证优化效果
5. **持续改进** - 迭代优化

### 性能优化方向

| 方向 | 方法 | 效果 |
|------|------|------|
| 算法优化 | 更高效的算法、数据结构 | 大幅提升 |
| 并行优化 | 多线程、多进程、分布式 | 显著提升 |
| 内存优化 | 减少内存占用、内存复用 | 资源节省 |
| I/O优化 | 异步I/O、批量处理 | 响应提升 |
| 缓存优化 | 结果缓存、预计算 | 延迟降低 |

### 模型调优方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 超参调优 | Grid Search、Bayesian Optimization | 效果提升 |
| 模型压缩 | 剪枝、量化、蒸馏 | 部署优化 |
| 数据增强 | 增加训练数据多样性 | 效果提升 |
| 集成学习 | 模型融合、Stacking | 效果提升 |
| 迁移学习 | 预训练模型微调 | 数据不足 |

## 常见场景

### 场景1：模型推理速度优化

**问题：** 模型推理延迟过高

**优化方向：**
1. 模型量化（FP32→FP16/INT8）
2. 批量推理
3. 模型剪枝
4. ONNX/TensorRT加速
5. 缓存机制

### 场景2：模型效果提升

**问题：** 模型效果达不到预期

**优化方向：**
1. 数据质量检查
2. 特征工程优化
3. 超参数调优
4. 模型结构调整
5. 集成学习
