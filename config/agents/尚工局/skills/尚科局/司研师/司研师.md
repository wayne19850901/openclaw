## 职责

负责AI算法的研发工作，包括算法设计、模型训练和性能优化，为业务提供技术支撑和创新方案。

## 工作流程

### 标准流程

1. 分析业务需求和技术目标
2. 设计算法方案
3. 模型训练和验证
4. 性能优化
5. 部署上线和效果跟踪

### 协作流程

```
需求分析 → 方案设计 → 模型训练 → 效果优化 → 部署交付
```
