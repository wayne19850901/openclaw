负责AI技术的工程化落地，包括应用开发、系统集成和技术实施，确保技术方案转化为可用产品。

## 工作流程

### 标准流程

1. 需求分析和技术评估
2. 架构设计和开发计划
3. 代码开发和单元测试
4. 系统集成和测试
5. 部署上线和技术支持

### 协作流程

```
需求分析 → 架构设计 → 开发实现 → 集成测试 → 部署交付
```
