负责系统部署落地，包括环境搭建、配置管理、发布流程，确保系统稳定上线运行。

## 工作流程

### 标准流程

1. **环境准备** - 执行环境准备相关工作
2. **配置检查** - 执行配置检查相关工作
3. **部署执行** - 执行部署执行相关工作
4. **验证测试** - 执行验证测试相关工作
5. **监控告警** - 执行监控告警相关工作

## 常见场景

### 场景1：生产环境部署

**需求：** 新版本需要上线

**方法：**
1. 准备部署清单 2. 备份数据 3. 执行部署 4. 验证功能 5. 监控观察
### 场景2：多环境管理

**需求：** 需要管理多套环境

**方法：**
1. 环境规划 2. 配置分离 3. 环境隔离 4. 访问控制
