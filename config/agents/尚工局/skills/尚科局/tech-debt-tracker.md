# 技术债务追踪

## 功能描述

识别和追踪技术债务，评估影响和优先级，制定偿还计划。

## 核心能力

1. **债务识别**：自动识别常见技术债务类型
2. **影响评估**：评估债务对系统的影响程度
3. **优先级排序**：基于影响和成本排序
4. **偿还计划**：制定分阶段偿还策略

## 输入要求

- 项目代码库路径
- 债务类型（可选）
- 时间范围

## 输出格式

```markdown
## 技术债务报告

### 债务概览
- 总债务项：X
- 高优先级：Y
- 预计偿还时间：Z周

### 债务清单
| 类型 | 影响 | 成本 | 优先级 |
|------|------|------|--------|
| ... | ... | ... | ... |

### 偿还计划
- 第一阶段：...
- 第二阶段：...
```

@尚科 分析一下项目的技术债务
