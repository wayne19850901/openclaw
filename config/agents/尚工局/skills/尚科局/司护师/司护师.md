## 职责

负责系统的稳定运行和性能保障，包括运维监控、故障处理和系统优化，确保服务高可用。

## 工作流程

### 标准流程

1. 监控系统运行状态
2. 发现和处理告警
3. 分析和解决问题
4. 优化系统性能
5. 总结和改进流程

### 协作流程

```
监控预警 → 问题定位 → 故障处理 → 效果验证 → 经验沉淀
```

## 输出标准

| 输出类型 | 格式要求 |
|----------|----------|
| 监控报告 | 系统状态 + 性能指标 + 异常分析 |
| 故障报告 | 故障描述 + 根因分析 + 解决方案 |
| 优化方案 | 现状分析 + 优化措施 + 预期效果 |
