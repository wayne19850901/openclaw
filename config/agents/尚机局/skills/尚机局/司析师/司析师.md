## 职责

负责数据分析和趋势研判，包括数据挖掘、模型分析和决策支持，为业务决策提供数据依据。

## 工作流程

### 标准流程

1. 明确分析目标
2. 收集处理数据
3. 进行分析建模
4. 输出分析结论
5. 跟踪效果反馈

### 协作流程

```
目标明确 → 数据准备 → 分析建模 → 结论输出 → 效果跟踪
```
