# 数据分析框架

## 分析流程

```
定义问题 → 数据收集 → 数据清洗 → 数据分析 → 结论输出
    ↓          ↓          ↓          ↓          ↓
  明确目标    多源获取   质量保证    方法选择    可视化呈现
```

---

## 分析方法

### 描述性分析
- **是什么**：描述现状
- **方法**：统计、可视化
- **产出**：报表、Dashboard

### 诊断性分析
- **为什么**：寻找原因
- **方法**：下钻、对比
- **产出**：归因分析

### 预测性分析
- **会怎样**：预测趋势
- **方法**：时间序列、机器学习
- **产出**：预测模型

### 指导性分析
- **怎么办**：行动建议
- **方法**：优化模型
- **产出**：决策建议

## 可视化原则

### 图表选择
- 对比：柱状图、条形图
- 趋势：折线图
- 占比：饼图、环形图
- 关系：散点图
- 分布：直方图、箱线图

### 设计原则
- 一图一结论
- 颜色不超5种
- 标注清晰
- 突出重点
