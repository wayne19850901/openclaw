# 异常检测

## 功能描述

自动检测数据中的异常值和异常模式，支持多种检测算法。

## 核心能力

1. **统计检测**：基于统计方法检测异常
2. **模式识别**：识别异常模式
3. **根因分析**：分析异常可能原因
4. **影响评估**：评估异常影响程度

## 输入要求

- 数据集或数据源
- 检测维度
- 敏感度设置

## 输出格式

```markdown
## 异常检测报告

### 检测概览
- 数据量：...
- 检测维度：...
- 异常数量：...

### 异常清单
| 时间 | 指标 | 异常值 | 正常范围 | 偏离程度 |
|------|------|--------|----------|----------|
| ... | ... | ... | ... | ... |

### 根因分析
- 可能原因：...
- 影响范围：...
- 建议处理：...
```

## 使用示例

```
@尚机 检测一下数据异常
```
