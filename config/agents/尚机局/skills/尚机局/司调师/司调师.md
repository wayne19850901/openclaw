## 职责

负责信息汇总和任务调度，包括资源协调、任务分配和进度跟踪，确保各项工作有序推进。

## 工作流程

### 标准流程

1. 收集任务和资源信息
2. 分析调度需求
3. 制定调度方案
4. 执行资源调配
5. 跟踪反馈并优化

### 协作流程

```
信息收集 → 需求分析 → 方案制定 → 执行调配 → 反馈优化
```

---

## 输出标准

| 输出类型 | 格式要求 |
|----------|----------|
| 调度方案 | 资源状态 + 调配计划 + 预期效果 |
| 任务分配 | 任务分解 + 责任人 + 时间节点 |
| 进度报告 | 完成情况 + 问题汇总 + 下步计划 |
