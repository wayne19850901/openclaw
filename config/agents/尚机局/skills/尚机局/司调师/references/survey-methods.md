# 调研方法指南

## 定量调研

### 问卷调查
**适用场景**：大规模数据收集

**设计要点**：
- 问题数量≤20个
- 避免引导性问题
- 选项互斥且穷尽
- 逻辑跳转合理

**投放渠道**：
- 问卷星/腾讯问卷
- 社群投放
- 定向推送

---

## 定性调研

### 用户访谈
**适用场景**：深度了解用户

**访谈流程**：
1. 建立信任（5分钟）
2. 开放提问（20分钟）
3. 深度挖掘（15分钟）
4. 总结确认（5分钟）

**提问技巧**：
- 开放式问题为主
- 追问"为什么"
- 避免诱导
- 倾听为主

### 焦点小组
**适用场景**：探索性研究

**组织要点**：
- 人数：6-8人
- 时长：1.5-2小时
- 主持人引导
- 记录+录像

---

## 混合方法

### 德尔菲法
**适用场景**：专家预测

**流程**：
1. 匿名征集意见
2. 汇总反馈
3. 多轮迭代
4. 达成共识

---

## 调研报告结构

```
1. 研究背景与目的
2. 研究方法说明
3. 样本概况
4. 主要发现
5. 结论与建议
6. 附录（问卷、访谈提纲等）
```
