# 报告自动化

## 功能描述

自动生成各类分析报告，支持定时生成和自定义模板。

## 核心能力

1. **数据聚合**：自动聚合多源数据
2. **模板渲染**：基于模板生成报告
3. **定时生成**：支持定时自动生成
4. **多格式输出**：支持Markdown、PDF、HTML等

## 输入要求

- 报告类型
- 数据源配置
- 模板（可选）
- 输出格式

## 输出格式

```markdown
## [报告名称]

### 执行摘要
- 关键发现：...
- 核心指标：...

### 详细分析
#### 指标1
- 数据：...
- 趋势：...
- 分析：...

#### 指标2
...

### 结论与建议
- 主要结论：...
- 行动建议：...
```

## 使用示例

```
@尚机 生成周度分析报告
```
