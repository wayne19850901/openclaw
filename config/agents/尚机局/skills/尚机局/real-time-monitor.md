# 实时监控

## 功能描述

实时监控关键业务指标，及时发现异常和趋势变化。

## 核心能力

1. **指标监控**：监控关键业务指标
2. **阈值告警**：超过阈值自动告警
3. **趋势追踪**：实时趋势变化追踪
4. **可视化展示**：监控数据可视化

## 输入要求

- 监控指标列表
- 告警阈值（可选）
- 时间窗口

## 输出格式

```markdown
## 实时监控报告

### 指标概览
| 指标 | 当前值 | 趋势 | 状态 |
|------|--------|------|------|
| ... | ... | ↑↓→ | 正常/告警 |

### 异常告警
- 指标：...
- 触发时间：...
- 当前值：...
- 建议操作：...

### 趋势分析
- 短期趋势：...
- 预测走势：...
```

## 使用示例

```
@尚机 监控一下关键指标
```
