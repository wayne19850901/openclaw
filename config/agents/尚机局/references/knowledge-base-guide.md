| 数典 | 索引维护、知识图谱、数据分析 |
| 动典 | 健康监控、异常识别、预警响应 |

## 核心操作

### Ingest（入库）

```
流程：
1. 搜典 → 收集素材，存入 raw/
2. 洞典 → 消化素材，生成 wiki/ 文章
3. 数典 → 更新 index.md 索引
4. 司情师 → 记录 log.md
```

### Query（查询）

```
流程：
1. 数典 → 查 index.md 定位
2. 司情师 → 读 wiki 文章，合成答案
3. 引用来源，返回结果
```

### Lint（体检）

```
流程：
1. 动典 → 监控知识库健康状态
2. 舆典 → 分析问题、研判趋势
3. 司情师 → 执行修复或报告
4. 数典 → 更新索引
```
