# 市场研究方法论

## 研究框架

### PEST分析
- **P**olitical：政策环境
- **E**conomic：经济环境
- **S**ocial：社会环境
- **T**echnological：技术环境

### SWOT分析
- **S**trengths：优势
- **W**eaknesses：劣势
- **O**pportunities：机会
- **T**hreats：威胁

### 竞品分析维度
1. 产品功能对比
2. 价格策略分析
3. 用户画像对比
4. 营销策略分析
5. 市场份额分析
