## 工作流程

### 产品设计流程

1. **需求调研** - 收集市场需求、客户反馈、竞品信息
2. **方案设计** - 制定产品要素（额度、期限、利率、还款方式）
3. **风控评估** - 与司风师协同评估产品风险
4. **定价测算** - 测算资金成本、风险成本、预期收益
5. **方案定稿** - 输出完整产品方案文档

### 产品要素清单

| 要素 | 说明 | 示例 |
|------|------|------|
| 产品名称 | 产品定位与命名 | 经营贷、消费贷、房贷 |
| 目标客群 | 适用客户类型 | 小微企业主、工薪族 |
| 贷款额度 | 最高/最低额度 | 10万-500万 |
| 贷款期限 | 可选期限范围 | 6个月-5年 |
| 利率范围 | 年化利率区间 | 3.5%-8% |
| 还款方式 | 等额本息/先息后本等 | 按月付息到期还本 |
| 担保方式 | 抵押/质押/信用 | 房产抵押 |

## 常见场景

### 场景1：设计小微企业经营贷产品

**输入：** 目标客群为年营业额100-1000万的小微企业主，需求是短期经营周转

**输出要点：**
- 额度：50-300万
- 期限：6-24个月
- 利率：4%-6%
- 还款：先息后本为主
- 担保：房产抵押或企业流水贷

### 场景2：优化现有产品

**输入：** 现有消费贷产品转化率低，需要优化

**输出：**
- 分析现有产品问题
- 提出优化建议（降低门槛、简化流程、调整定价）
- A/B测试方案
