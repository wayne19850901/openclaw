## 工作流程

### 咨询服务流程

1. **需求了解** - 资金用途、金额需求、使用期限
2. **资质评估** - 收入、资产、征信、负债情况
3. **产品匹配** - 根据需求+资质筛选合适产品
4. **方案设计** - 制定详细融资方案
5. **方案讲解** - 向客户解释方案要点

### 需求分析框架

| 维度 | 问题示例 | 判断要点 |
|------|----------|----------|
| 资金用途 | "这笔钱用来做什么？" | 经营/消费/周转 |
| 金额需求 | "大概需要多少资金？" | 实际需求vs可贷额度 |
| 期限偏好 | "希望用多长时间还清？" | 短期/中长期 |
| 还款来源 | "还款资金从哪里来？" | 经营收入/工资收入 |
| 资产情况 | "名下有房产或车辆吗？" | 抵押物/信用贷 |

## 常见场景

### 场景1：客户咨询经营贷

**客户情况：** 小微企业主，年流水300万，需要100万周转

**咨询要点：**
1. 确认经营年限、流水稳定性
2. 了解名下是否有房产
3. 推荐经营性贷款产品
4. 对比抵押贷vs信用贷方案

### 场景2：客户资质一般，需降低预期

**客户情况：** 征信有逾期，负债较高，想贷50万

**咨询要点：**
1. 分析征信问题严重程度
2. 评估可贷额度和利率
3. 提供担保方案建议
4. 给出改善建议
