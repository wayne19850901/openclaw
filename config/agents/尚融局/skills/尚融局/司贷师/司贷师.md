## 工作流程

### 标准流程

1. 了解客户/业务需求
2. 分析现有产品和流程
3. 调用对应典级服务
4. 输出方案或建议
5. 跟进实施效果

### 协作流程

```
接收任务 → 需求分析 → 调用典级 → 输出方案 → 反馈优化
```

---

## 输出标准

| 输出类型 | 格式要求 |
|----------|----------|
| 产品方案 | 结构化文档，包含目标、设计、风控、定价 |
| 咨询建议 | 针对性方案，包含分析、推荐、理由 |
| 流程优化 | 流程图 + 改进点 + 预期效果 |
