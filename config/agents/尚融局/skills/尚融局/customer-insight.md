# 客户洞察分析

## 功能描述

分析客户行为数据，识别需求模式和购买意向，为销售和营销决策提供数据支持。

## 核心能力

1. **行为模式识别**：分析客户购买历史、浏览行为、互动记录
2. **需求预测**：基于行为数据预测客户潜在需求
3. **价值分层**：客户价值评估与分层管理
4. **流失预警**：识别高风险流失客户

## 输入要求

- 客户基础数据（可选）
- 交易历史记录
- 行为日志数据

## 输出格式

```markdown
## 客户洞察报告

### 客户画像
- 核心特征：...
- 行为偏好：...

### 需求分析
- 当前需求：...
- 潜在需求：...

### 建议行动
- 营销策略：...
- 服务优化：...
```

## 使用示例

```
@尚融 分析一下这个客户群体的购买行为
```
