## 工作流程

### 获客流程

1. **目标定义** - 明确目标客户画像
2. **渠道选择** - 选择合适获客渠道
3. **投放执行** - 执行获客活动
4. **效果追踪** - 监控转化效果
5. **优化迭代** - 优化获客策略

### 获客渠道

| 渠道类型 | 适用场景 | 优缺点 |
|----------|----------|--------|
| 线上广告 | 大规模获客 | 成本可控、精准投放 |
| 异业合作 | 精准获客 | 质量高、转化好 |
| 老客转介绍 | 低成本获客 | 信任度高、转化高 |
| 地推活动 | 本地获客 | 面对面、效果好 |
| 社群运营 | 长期培育 | 成本低、需时间 |

### 销售话术库

**开场白：**
```
您好，我是XX金融的专属顾问[姓名]，专注为您解决资金周转问题。
请问您近期有资金方面的需求吗？
```

**需求挖掘：**
```
请问：
1. 您这笔资金主要用途是什么？（经营/消费/周转）
2. 大概需要多少金额？
3. 期望多久能拿到资金？
4. 您名下有房产或车辆吗？
```

**异议处理：**
```
客户：利息太高了
回应：我理解您的顾虑。其实我们的综合成本在行业内是很有竞争力的，
      而且审批快、放款快，能帮您抓住商机。您觉得时间成本重要吗？
```

**促成话术：**
```
根据您的情况，我为您匹配了最优方案：额度XX万，月息X厘，X天放款。
您看这个方案可以吗？我们先把资料准备起来？
```

## 常见场景

### 场景1：开拓新渠道

**需求：** 需要开拓房产中介渠道

**方案要点：**
1. 筛选目标中介机构
2. 制定合作方案（返佣、培训）
3. 建立对接机制
4. 跟踪效果并优化

### 场景2：提升转化率

**问题：** 咨询客户多，但转化率低

**优化方案：**
1. 分析流失环节
2. 优化跟进流程
3. 强化话术培训
4. 建立激励机制
