## 工作流程

### 客户维护流程

1. **客户分层** - 按价值和潜力分层
2. **制定计划** - 不同层级不同维护策略
3. **执行沟通** - 定期联系、节日问候
4. **需求挖掘** - 了解新需求、复购机会
5. **效果评估** - 评估维护效果

### 客户分层标准

| 层级 | 标准 | 维护策略 | 维护频率 |
|------|------|----------|----------|
| VIP客户 | 贷款>100万，复购2次+ | 专属顾问、优先服务 | 每周联系 |
| 优质客户 | 贷款>50万，复购1次 | 定期回访、节日礼品 | 每月联系 |
| 普通客户 | 贷款<50万 | 节日问候、活动通知 | 季度联系 |
| 潜在客户 | 咨询未成交 | 定期跟进、需求唤醒 | 按需联系 |

### 维护话术

**贷后回访：**
```
您好，XX先生/女士，我是您的专属顾问[姓名]。
想了解一下您的资金使用情况，有什么需要帮助的吗？
```

**节日问候：**
```
XX节快乐！感谢您对我们的信任和支持。
祝您节日愉快，生意兴隆！有需要随时联系我。
```

**续贷提醒：**
```
您好，您的贷款即将到期，需要续贷的话可以提前准备资料。
我这边可以帮您预留额度，您看需要吗？
```

## 常见场景

### 场景1：VIP客户维护

**客户情况：** 企业主，贷款200万，还款良好

**维护策略：**
1. 专属顾问一对一服务
2. 定期上门拜访
3. 生日/节日礼品
4. 新产品优先通知
5. 续贷提前准备

### 场景2：客户满意度下降

**问题：** 客户投诉审批慢，满意度下降

**处理方式：**
1. 诚恳道歉，了解诉求
2. 解释原因，说明改进措施
3. 给予适当补偿（利率优惠等）
4. 持续跟进，恢复信任
