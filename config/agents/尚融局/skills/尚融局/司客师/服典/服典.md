## 工作流程

### 问题处理流程

1. **接收问题** - 记录客户问题和诉求
2. **分类判断** - 判断问题类型和紧急程度
3. **协调处理** - 协调相关部门解决
4. **反馈结果** - 向客户反馈处理结果
5. **满意度确认** - 确认客户满意度

### 问题分类与时效

| 问题类型 | 说明 | 响应时效 | 解决时效 |
|----------|------|----------|----------|
| 紧急问题 | 放款失败、系统故障 | 10分钟 | 2小时 |
| 一般问题 | 流程咨询、材料要求 | 30分钟 | 24小时 |
| 投诉问题 | 服务投诉、利益纠纷 | 1小时 | 3天 |
| 建议反馈 | 服务建议、产品建议 | 24小时 | 7天 |

### 服务话术

**问题接待：**
```
您好，请问有什么可以帮您？
请详细描述一下您遇到的问题，我来帮您处理。
```

**进度查询：**
```
您的申请目前处于[审批中]，预计[X]个工作日内完成。
审批通过后我们会第一时间通知您。
```

**投诉处理：**
```
非常抱歉给您带来不便，我理解您的心情。
请您详细说明情况，我会尽快帮您处理并给您满意的答复。
```

**问题解决：**
```
您的问题已经处理完成，[说明处理结果]。
请问您对这个处理结果满意吗？还有其他需要帮助的吗？
```

## 常见场景

### 场景1：客户投诉审批慢

**处理步骤：**
1. 倾听客户诉求，表达理解
2. 查询审批进度，解释原因
3. 协调加快审批
4. 给予合理预期
5. 跟进反馈结果

### 场景2：放款失败客户不满

**处理步骤：**
1. 诚恳道歉，安抚情绪
2. 核实失败原因
3. 提供解决方案
4. 协助重新申请
5. 适当补偿（如费用减免）
