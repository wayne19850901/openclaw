# CRM数据同步

## 功能描述

同步和分析CRM系统数据，确保客户信息一致性，支持跨系统数据整合。

## 核心能力

1. **数据同步**：多源CRM数据整合
2. **数据清洗**：去重、标准化、补全
3. **一致性检查**：识别数据冲突和不一致
4. **同步日志**：记录同步历史和变更

## 输入要求

- CRM数据源配置
- 同步规则（可选）
- 数据映射关系

## 输出格式

```markdown
## CRM同步报告

### 同步概览
- 数据源：...
- 同步记录数：...
- 新增/更新/删除：...

### 数据质量
- 完整性：...
- 准确性：...
- 一致性：...

### 问题清单
- 待处理冲突：...
- 建议操作：...
```
