## 职责

负责贷款逾期后的催收处理，包括催收策略制定、客户谈判、风险处置，最大限度挽回损失、控制不良率。

## 工作流程

### 催收流程

1. **逾期识别** - 系统监控，识别逾期客户
2. **原因分析** - 了解逾期原因（忘记/困难/恶意）
3. **策略制定** - 根据逾期阶段和原因制定策略
4. **执行催收** - 电话/上门/法律手段
5. **结果跟踪** - 记录催收结果，持续跟进

### 逾期阶段与策略

| 阶段 | 逾期天数 | 策略 | 手段 |
|------|----------|------|------|
| 提醒期 | 1-7天 | 温馨提醒 | 短信、电话提醒 |
| 早期 | 8-30天 | 积极沟通 | 电话催收、了解原因 |
| 中期 | 31-90天 | 施压谈判 | 上门、亲属联系人 |
| 后期 | 91-180天 | 法律手段 | 律师函、起诉 |
| 坏账期 | >180天 | 资产处置 | 打包转让、核销 |

### 催收话术

**早期催收话术：**
```
您好，我是XX金融的客服，您本月贷款已逾期X天，请尽快还款。
请问是什么原因导致逾期？我们这边可以帮您安排还款方案。
```

**中期催收话术：**
```
您的贷款已逾期X天，已产生违约金X元，并会上报征信。
建议您尽快处理，避免影响信用记录。
```

**谈判还款方案：**
```
理解您的困难，我们可以协商分期还款方案：
方案1：先还本金，利息延后
方案2：分期偿还，每月X元
您看哪个方案更适合您？
```

## 常见场景

### 场景1：客户声称困难，请求延期

**处理方式：**
1. 核实困难情况（收入证明、困难证明）
2. 评估还款意愿和还款能力
3. 制定可行的分期方案
4. 签署还款协议，持续跟进

### 场景2：客户失联，电话打不通

**处理方式：**
1. 尝试紧急联系人
2. 查询最新联系方式
3. 必要时上门核实
4. 无法联系转为法律程序
