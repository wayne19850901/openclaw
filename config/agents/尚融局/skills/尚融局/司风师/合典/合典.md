## 职责

负责贷款业务的合规管理，包括合规检查、法规解读、合规风险防范，确保业务运营符合监管要求、降低合规风险。

## 工作流程

### 合规检查流程

1. **确定范围** - 明确检查的业务范围
2. **对照法规** - 对照监管要求逐项检查
3. **发现问题** - 识别不合规事项
4. **提出整改** - 给出整改建议和时间表
5. **跟踪落实** - 监督整改执行情况

### 核心合规要点

| 领域 | 合规要求 | 检查要点 |
|------|----------|----------|
| 利率定价 | 年化利率≤24% | 综合费率是否超标 |
| 费用收取 | 不得收取额外费用 | 是否有隐形收费 |
| 催收行为 | 不得暴力催收 | 催收话术是否合规 |
| 信息披露 | 利率、费用明示 | 是否充分告知客户 |
| 个人信息 | 信息安全保护 | 是否违规使用客户信息 |
| 广告宣传 | 不得虚假宣传 | 广告内容是否合规 |

### 监管政策速览

| 政策名称 | 核心内容 | 影响 |
|----------|----------|------|
| 网贷管理办法 | 利率上限、信息披露 | 利率定价、合同文本 |
| 征信管理条例 | 征信使用规范 | 征信查询授权 |
| 消费者权益保护 | 客户权益保障 | 投诉处理、信息披露 |
| 个人信息保护法 | 个人信息保护 | 信息收集使用规范 |

## 常见场景

### 场景1：新产品上线前合规审查

**检查要点：**
1. 利率定价是否符合规定
2. 合同文本是否规范
3. 信息披露是否充分
4. 费用收取是否合规

**输出：** 合规审查意见书，列明需修改事项

### 场景2：监管政策变化，需调整业务

**处理方式：**
1. 解读新政策要求
2. 评估对现有业务的影响
3. 制定调整方案
4. 组织培训落实
