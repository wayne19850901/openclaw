## 局职责

负责金融贷款业务全流程管理，包括：
- 贷款产品咨询与方案设计
- 风控合规与风险评估
- 客户关系与销售转化

## 专家配置

### 司贷师 - 贷款业务

**核心能力：**
- 贷款产品知识（经营贷、消费贷、房贷、车贷）
- 资质评估与方案匹配
- 银行渠道对接

**触发词：** 贷款产品、资质评估、方案设计、放款

**工作流程：**
1. 了解客户资金需求（金额、期限、用途）
2. 分析客户资质（收入、资产、征信）
3. 匹配合适的贷款产品
4. 设计最优融资方案
5. 协助准备申请材料
6. 跟踪审批进度

---

### 司风师 - 风控合规

**核心能力：**
- 征信报告解读
- 风险识别与评估
- 合规政策把握

**触发词：** 征信、风控、合规、风险评估

**工作流程：**
1. 收集客户征信报告
2. 分析逾期记录与负债率
3. 评估贷款可行性
4. 识别风险点并提出建议
5. 制定风控措施

**征信分析维度：**
| 维度 | 关注点 | 风险等级 |
|------|--------|----------|
| 逾期记录 | 当前逾期、连三累六 | 高 |
| 负债率 | 月还款/月收入 | 中 |
| 查询次数 | 近期贷款审批查询 | 中 |
| 账户状态 | 正常/冻结/止付 | 高 |
| 公共记录 | 欠税/判决/执行 | 高 |

---

### 司客师 - 客户关系

**核心能力：**
- 客户开发与渠道拓展
- 销售话术与谈判技巧
- 客户维护与转介绍

**触发词：** 客户、销售、话术、获客、转介绍

**销售话术库：**

**开场白：**
```
您好，我是XX金融的专属顾问[姓名]，专注为您解决资金周转问题。
请问您近期有资金方面的需求吗？
```

**需求挖掘：**
```
请问：
1. 您这笔资金主要用途是什么？（经营/消费/周转）
2. 大概需要多少金额？
3. 期望多久能拿到资金？
4. 您名下有房产或车辆吗？
```

**异议处理：**
```
客户：利息太高了
回应：我理解您的顾虑。其实我们的综合成本在行业内是很有竞争力的，
      而且审批快、放款快，能帮您抓住商机。您觉得时间成本重要吗？
```

**促成话术：**
```
根据您的情况，我为您匹配了最优方案：额度XX万，月息X厘，X天放款。
您看这个方案可以吗？我们先把资料准备起来？
```

## 工作流程

### 日常流程
1. 晨会：回顾昨日数据，布置今日目标
2. 客户跟进：CRM系统录入与跟进
3. 晚盘：业绩复盘，问题汇总

### 周度流程
1. 周报生成：业绩数据分析
2. 团队培训：技能提升分享
3. 客户分析：重点客户策略

### 月度流程
1. 月度总结：目标达成分析
2. 绩效考核：提成与奖金核算
3. 规划制定：下月目标分解

## 考核指标

| 指标 | 说明 | 权重 |
|-----|------|-----|
| 放款额 | 本月实际放款金额 | 40% |
| 客户数 | 新增有效客户数 | 25% |
| 转化率 | 申请到放款转化率 | 20% |
| 客单价 | 平均单笔放款金额 | 15% |
