# 销售预测分析

## 功能描述

基于历史销售数据、市场趋势和季节性因素，预测未来销售业绩，支持财务规划和资源分配。

## 核心能力

1. **趋势分析**：识别销售趋势和周期性模式
2. **季节调整**：考虑季节性因素影响
3. **情景模拟**：乐观/中性/悲观三种情景预测
4. **准确度评估**：历史预测准确率追踪

## 输入要求

- 历史销售数据
- 市场环境信息（可选）
- 预测周期

## 输出格式

```markdown
## 销售预测报告

### 预测结果
- 预测周期：...
- 预测销售额：...
- 置信区间：...

### 情景分析
- 乐观情景：...
- 中性情景：...
- 悲观情景：...

### 关键驱动因素
- 正面因素：...
- 风险因素：...
```
