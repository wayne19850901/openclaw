# 财务计算公式库

## 基础指标

### ROI (投资回报率)
```
ROI = (收益 - 成本) / 成本 × 100%
```

### 利润率
```
毛利率 = (营收 - 成本) / 营收 × 100%
净利率 = 净利润 / 营收 × 100%
```

### 现金流
```
经营现金流 = 营业收入 - 营业成本 - 税费
自由现金流 = 经营现金流 - 资本支出
```

## 贷款相关

### 月供计算 (等额本息)
```
月供 = [本金 × 月利率 × (1+月利率)^还款月数] / [(1+月利率)^还款月数 - 1]
```

### 实际利率 (IRR)
```
IRR: 使NPV=0的折现率
```

### 综合成本
```
综合成本 = 利息 + 手续费 + 评估费 + 其他费用
```

## 风险评估

### 信用评分要素
1. 还款历史 (35%)
2. 负债水平 (30%)
3. 信用历史长度 (15%)
4. 新开账户 (10%)
5. 信用组合 (10%)
