## 使用说明

1. **选择模板**：根据妃子职责选择对应专属章节
2. **填充内容**：将专属章节插入标准结构的第三章
3. **定期更新**：每次重要任务后更新相关章节
4. **知识贡献**：创建新知识后记录在第六章
