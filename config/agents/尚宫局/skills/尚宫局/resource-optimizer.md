# 资源优化器

## 功能描述

分析资源使用情况，优化资源分配，提高使用效率。

## 核心能力

1. **使用分析**：分析资源使用率和分配情况
2. **瓶颈识别**：识别资源瓶颈和浪费
3. **优化建议**：提供资源优化方案
4. **成本评估**：评估优化成本和收益

## 输入要求

- 资源清单
- 使用数据
- 预算约束（可选）

## 输出格式

```markdown
## 资源优化报告

### 使用概览
| 资源 | 分配量 | 使用率 | 状态 |
|------|--------|--------|------|
| ... | ... | X% | 充足/紧张/浪费 |

### 优化建议
- 节省项：...
- 增加项：...
- 预计节省：X%

### 实施计划
- 第一阶段：...
- 第二阶段：...
```

## 使用示例

```
@尚宫 优化一下资源分配
```
