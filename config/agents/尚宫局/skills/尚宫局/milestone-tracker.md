# 里程碑追踪

## 功能描述

追踪项目里程碑完成情况，及时发现延期风险。

## 核心能力

1. **进度追踪**：实时追踪里程碑进度
2. **延期预警**：提前预警延期风险
3. **依赖分析**：分析里程碑间依赖关系
4. **甘特图生成**：可视化项目时间线

## 输入要求

- 项目计划
- 里程碑列表
- 当前进度数据

## 输出格式

```markdown
## 里程碑追踪报告

### 进度概览
| 里程碑 | 计划日期 | 实际日期 | 状态 |
|--------|----------|----------|------|
| ... | ... | ... | 完成/进行中/延期 |

### 延期风险
- 风险里程碑：...
- 预计延期：...
- 影响范围：...

### 建议行动
- 加速措施：...
- 资源调整：...
```

## 使用示例

```
@尚宫 追踪一下项目里程碑
```
