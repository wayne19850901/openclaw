## 职责

负责项目的全生命周期管理，包括项目规划、进度把控和资源统筹，确保项目按时按质完成。

## 工作流程

### 标准流程

1. 项目启动和规划
2. 资源分配和调度
3. 执行监控和调整
4. 问题处理和协调
5. 项目收尾和复盘

### 协作流程

```
项目启动 → 规划设计 → 执行监控 → 调整优化 → 收尾复盘
```

## 输出标准

| 输出类型 | 格式要求 |
|----------|----------|
| 项目计划 | 目标范围 + 进度计划 + 资源需求 |
| 进度报告 | 完成情况 + 风险问题 + 下步计划 |
| 复盘报告 | 项目总结 + 经验教训 + 改进建议 |
