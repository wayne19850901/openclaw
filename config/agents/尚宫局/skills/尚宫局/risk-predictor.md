# 风险预测器

## 功能描述

基于项目数据和历史模式，预测潜在风险和问题。

## 核心能力

1. **风险识别**：识别项目潜在风险
2. **概率评估**：评估风险发生概率
3. **影响分析**：分析风险影响程度
4. **缓解方案**：提供风险缓解策略

## 输入要求

- 项目信息
- 历史风险数据（可选）
- 风险偏好

## 输出格式

```markdown
## 风险预测报告

### 风险概览
| 风险 | 概率 | 影响 | 等级 |
|------|------|------|------|
| ... | 高/中/低 | 高/中/低 | P0-P3 |

### 详细分析
#### 风险1
- 描述：...
- 触发条件：...
- 影响范围：...

### 缓解方案
- 预防措施：...
- 应急预案：...
```

## 使用示例

```
@尚宫 预测一下项目风险
```
