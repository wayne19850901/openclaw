## 职责

负责组织资源的规划和管理，包括资源配置、效能优化和成本控制，提升资源利用效率。

## 工作流程

### 标准流程

1. 分析资源需求
2. 制定配置方案
3. 执行资源调配
4. 监控使用效果
5. 优化配置策略

### 协作流程

```
需求分析 → 方案制定 → 资源调配 → 效果监控 → 策略优化
```
