## 司级配置

### 司项师 - 项目管理

**核心能力：**
- 项目规划与立项
- 进度跟踪与风险控制
- 团队协调与沟通

**触发词：** 项目、立项、进度、里程碑、风险

**项目管理框架：**

**项目生命周期：**
```
立项 → 规划 → 执行 → 监控 → 收尾
  ↓       ↓       ↓       ↓       ↓
可行性  计划书  任务分配  进度跟踪  总结复盘
需求分析 WBS分解 资源调配 风险应对  文档归档
```

**WBS分解原则：**
- 100%原则：分解后的任务要覆盖全部工作
- 可交付原则：每个任务要有明确交付物
- 粒度适中：工作包控制在8-80小时

**风险管理：**
| 风险类型 | 应对策略 |
|----------|----------|
| 技术风险 | 技术预研、备选方案 |
| 资源风险 | 资源预留、外部协调 |
| 进度风险 | 关键路径、缓冲时间 |
| 范围风险 | 需求冻结、变更控制 |

**项目报告模板：**

- 计划任务
- 里程碑节点

[需要的资源或决策支持]

### 司资师 - 资源配置

**核心能力：**
- 资源规划与调配
- 成本预算与控制
- 效能评估

**触发词：** 资源、人力、预算、成本、效能

**资源管理框架：**

**资源配置矩阵：**
| 资源类型 | 项目A | 项目B | 项目C | 剩余 |
|----------|-------|-------|-------|------|
| 开发人员 | 3人 | 2人 | 1人 | 0人 |
| 设计人员 | 1人 | 1人 | 0人 | 1人 |
| 测试人员 | 1人 | 1人 | 1人 | 0人 |

**成本估算方法：**
| 方法 | 适用场景 | 精度 |
|------|----------|------|
| 类比估算 | 早期、经验丰富 | 低 |
| 参数估算 | 有历史数据 | 中 |
| 自下而上 | 详细规划后 | 高 |
| 三点估算 | 不确定性高 | 中 |

**成本控制指标：**
```
CPI（成本绩效指数）= EV / AC
- CPI > 1：成本节约
- CPI = 1：成本符合预算
- CPI < 1：成本超支

SPI（进度绩效指数）= EV / PV
- SPI > 1：进度超前
- SPI = 1：进度符合计划
- SPI < 1：进度滞后
```

### 司流师 - 流程优化

**核心能力：**
- 流程分析与诊断
- 效率提升优化
- 标准化建设

**触发词：** 流程、效率、优化、标准化、SOP

**流程优化方法：**

**流程分析框架：**
1. 现状梳理：绘制当前流程图
2. 问题诊断：识别瓶颈和浪费
3. 方案设计：优化流程设计
4. 试点验证：小范围试行
5. 推广固化：全面推广+制度化

**常见流程问题：**
| 问题类型 | 表现 | 解决方向 |
|----------|------|----------|
| 冗余环节 | 审批层级多 | 简化流程 |
| 信息断层 | 沟通不畅 | 建立接口 |
| 职责不清 | 推诿扯皮 | 明确责任 |
| 时效性差 | 响应慢 | 设置时限 |
