## 核心操作

### Ingest（入库）

```
操作流程：
1. 原始素材 → raw/对应主题/
2. 整理消化 → wiki/对应主题/
3. 更新索引 → wiki/index.md
4. 记录日志 → wiki/log.md
```

### Query（查询）

```
操作流程：
1. 查 index.md 定位
2. 读 wiki 文章
3. 合成答案 + 引用来源
```

### Lint（体检）

```
检查项：
- 索引完整性
- 链接有效性
- 孤立页面
- 事实矛盾
```
