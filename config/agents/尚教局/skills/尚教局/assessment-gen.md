# 测评生成

## 功能描述

根据课程内容和学习目标，自动生成各类测评题目。

## 核心能力

1. **题型多样**：选择题、填空题、简答题等
2. **难度分层**：基础/中等/困难多难度
3. **知识点覆盖**：确保知识点全面覆盖
4. **答案生成**：自动生成标准答案和解析

## 输入要求

- 课程内容或知识点
- 题目数量
- 难度要求

## 输出格式

```markdown
## 测评题目

### 选择题（基础）
1. 题目内容...
   A. ...
   B. ...
   C. ...
   D. ...
   
   答案：A
   解析：...

### 填空题（中等）
1. 题目内容____...
   
   答案：...
   解析：...
```

## 使用示例

```
@尚教 为这个知识点生成10道测试题
```
