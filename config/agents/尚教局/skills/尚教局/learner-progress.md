# 学习进度追踪

## 功能描述

追踪学员学习进度，分析学习效果，提供个性化建议。

## 核心能力

1. **进度可视化**：学习进度图表化展示
2. **效果评估**：学习效果量化评估
3. **瓶颈识别**：识别学习困难和瓶颈
4. **个性化建议**：基于进度提供学习建议

## 输入要求

- 学员ID或学习记录
- 课程信息
- 时间范围

## 输出格式

```markdown
## 学习进度报告

### 进度概览
- 完成率：X%
- 学习时长：Y小时
- 掌握知识点：Z个

### 学习曲线
- 进度趋势：...
- 效率变化：...

### 建议行动
- 重点复习：...
- 建议学习：...
```

## 使用示例

```
@尚教 查看这个学员的学习进度
```
