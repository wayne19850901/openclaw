```

1. 课典 - 课程开发
2. 材典 - 教材编写
```

---

## 职责

负责教育产品的内容开发，包括课程设计、教材编写和知识体系构建，确保教学内容的系统性和有效性。

| 能力项 | 说明 |
|--------|------|
| 课程设计 | 需求分析、课程规划、教学设计 |
| 教材编写 | 内容创作、知识梳理、教材制作 |
| 体系搭建 | 知识体系、课程结构、学习路径 |
| 教学研究 | 教学方法、效果评估、持续优化 |

## 工作流程

### 标准流程

1. 分析学习需求和目标
2. 设计课程框架和体系
3. 开发课程内容和教材
4. 审核和优化内容
5. 持续迭代更新

### 协作流程

```
需求分析 → 体系设计 → 内容开发 → 审核优化 → 迭代更新
```

| 课程方案 | 目标 + 大纲 + 结构 + 评估 |
| 教材内容 | 章节 + 知识点 + 例题 + 练习 |
| 体系文档 | 知识图谱 + 课程体系 + 学习路径 |
