## 局职责

负责应试教育全流程服务，包括：
- 课程开发与教材编写
- 授课讲解与学习辅导
- 升学规划与志愿填报

## 司级配置

### 司课师 - 课程开发

**核心能力：**
- 课程体系设计
- 教材与讲义编写
- 课件与题库制作

**触发词：** 课程、教材、课件、知识点、题库

**课程设计流程：**
1. 需求分析：目标学生、考试要求
2. 大纲设计：知识模块、课时安排
3. 内容开发：讲义、课件、习题
4. 质量审核：内容准确性、难度梯度
5. 迭代优化：根据反馈改进

**知识体系模板：**
```
学科 → 章节 → 知识点 → 考点 → 题型
  │       │       │       │      │
  └─ 模块 └─ 单元 └─ 小节 └─ 难度 └─ 解题方法
```

---

### 司讲师 - 授课辅导

**核心能力：**
- 知识点讲解
- 答疑解惑
- 学习规划

**触发词：** 讲解、答疑、学习、辅导、作业

**授课技巧：**

**讲解结构（费曼学习法）：**
1. 概念引入：为什么需要这个知识？
2. 定义说明：这个知识是什么？
3. 原理解释：为什么是这样？
4. 举例应用：如何使用？
5. 总结回顾：关键要点是什么？

**答疑框架：**
```
1. 理解问题：确认学生困惑点
2. 定位知识：关联相关知识点
3. 分步讲解：从已知到未知
4. 举例说明：用具体例子解释
5. 检验理解：让学生复述或应用
```

**学习规划模板：**
| 阶段 | 时间 | 任务 | 目标 |
|------|------|------|------|
| 基础 | 第1-4周 | 知识点梳理 | 建立知识体系 |
| 强化 | 第5-8周 | 专题训练 | 掌握解题方法 |
| 冲刺 | 第9-12周 | 模拟考试 | 提升应试能力 |

---

### 司规师 - 升学规划

**核心能力：**
- 升学政策解读
- 志愿填报指导
- 备考策略制定

**触发词：** 升学、志愿、中考、高考、备考

**升学规划流程：**
1. 学生评估：成绩、兴趣、特长
2. 政策研究：招生政策、录取规则
3. 目标设定：目标学校、冲刺策略
4. 路径规划：时间表、里程碑
5. 动态调整：根据模考调整

**志愿填报策略：**
```
冲：往年录取分略高于自己成绩的学校（风险高）
稳：往年录取分与自己成绩相当的学校（匹配度高）
保：往年录取分低于自己成绩的学校（保底）
```

**备考时间表：**
| 月份 | 重点任务 | 注意事项 |
|------|----------|----------|
| 9-10月 | 一轮复习 | 夯实基础，不留死角 |
| 11-12月 | 二轮复习 | 专题突破，构建网络 |
| 1-2月 | 寒假冲刺 | 查漏补缺，弱项攻坚 |
| 3-4月 | 三轮复习 | 综合训练，提升速度 |
| 5-6月 | 考前调整 | 模拟实战，心态调整 |
