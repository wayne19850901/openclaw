## 职责

负责教学授课和知识传递，包括课程讲授、学员辅导和学习服务，确保学习效果和学员满意度。

## 工作流程

### 标准流程

1. 准备课程内容
2. 开展教学活动
3. 辅导学员学习
4. 评估学习效果
5. 收集反馈优化

### 协作流程

```
课程准备 → 教学实施 → 学员辅导 → 效果评估 → 反馈优化
```

| 辅导记录 | 问题汇总 + 辅导内容 + 效果跟踪 |
| 服务报告 | 学员反馈 + 服务记录 + 改进建议 |
