# 自适应内容生成

## 功能描述

根据学员当前水平和学习风格，生成个性化的学习内容。

## 核心能力

1. **水平评估**：评估学员当前知识水平
2. **风格适配**：匹配学员学习风格偏好
3. **难度调节**：动态调整内容难度
4. **内容生成**：生成个性化学习材料

## 输入要求

- 学员信息或水平评估
- 学习目标
- 内容类型偏好

## 输出格式

```markdown
## 自适应学习内容

### 学员画像
- 当前水平：...
- 学习风格：...
- 目标水平：...

### 生成内容
- 标题：...
- 难度等级：...
- 内容：...

### 学习建议
- 预计学习时间：...
- 配套练习：...
```

## 使用示例

```
@尚教 为这个学员生成个性化学习内容
```
