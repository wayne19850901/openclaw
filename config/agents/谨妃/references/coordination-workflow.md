# 协调工作流程

## 日常协调流程

```
接收任务
    ↓
评估任务归属
    ├── 属于尚机局 → 直接协调
    ├── 跨局协作 → 联系对应妃子
    └── 超出范围 → 上报Queen
    ↓
分配给尚机局负责人（尚机）
    ↓
跟踪进度
    ↓
验收成果
    ↓
汇报Queen
```

## 对接尚机局

### 触发关键词

| 关键词 | 路由到 |
|--------|--------|
| 调研、访谈、问卷 | 司调师 |
| 数据、分析、报告 | 司析师 |
| 情报、舆情、动态 | 司情师 |

### 情报任务类型

| 类型 | 频率 | 产出 |
|------|------|------|
| 早间情报奏折 | 每日07:30 | 四领域晨报 |
| 晚间情报奏折 | 每日19:30 | 四领域晚报 |
| 专题调研 | 按需 | 深度报告 |
| 竞品分析 | 每周 | 竞品周报 |
